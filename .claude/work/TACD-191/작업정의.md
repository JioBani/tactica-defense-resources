# TACD-191: [전장 - 유닛] 능력치 정보 UI

> 상위 에픽: TACD-80 [전장] 유닛
> 관련 일감: TACD-122 [전장 - 유닛] 스탯 (완료)

---

## 작업 범위

상태이상 표시(TACD-195)를 **제외**하고 아래 하위 일감을 작업한다.

| 키 | 제목 | 설명 |
|---|---|---|
| TACD-197 | UI 임시 디자인 작성 | 패널 레이아웃과 영역 배치 확정 |
| TACD-192 | UI 제작 | 임시 디자인 기반 패널 프리팹 제작, 닫기 버튼, 스탯 행 프리팹 |
| TACD-193 | 유닛 정보 연결 | 유닛 이름, 일러스트, 성급 데이터 바인딩 |
| TACD-194 | 스탯 연결 | 스탯 목록(아이콘 + 스탯명 + 최종값) 데이터 바인딩 |
| TACD-196 | 유닛 선택/해제로 패널 열기·닫기 | 유닛 클릭 시 패널 표시, X 버튼 클릭 시 닫기 |

---

## 작업 내용

1. **능력치 정보 패널 UI** — 유닛을 선택(클릭/탭)하면 해당 유닛의 능력치 패널을 표시한다
2. **유닛 정보 표시** — 이름, 일러스트, 성급
3. **스탯 목록 표시** — 14종 능력치를 아이콘 + 스탯명 + 최종값 형태로 나열
4. **패널 열기/닫기** — 유닛 클릭 시 패널 열기, X 버튼 클릭 시 닫기

---

## 완료 정의

- [ ] StatInfoPanel UI 프리팹이 생성되어 있다
- [ ] 유닛 선택 시 해당 유닛의 이름, 일러스트, 성급이 표시된다
- [ ] 유닛 선택 시 해당 유닛의 14종 스탯이 아이콘 + 스탯명 + 최종값 형태로 표시된다
- [ ] X 버튼 클릭 시 패널이 닫힌다
