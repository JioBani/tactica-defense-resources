## 2. TACD-197: UI 임시 디자인

- HTML/CSS 파일로 패널 레이아웃 목업을 제작한다
- 패널 구성 영역:
  - **상단**: 닫기(X) 버튼 (오른쪽 상단)
  - **유닛 정보 영역**: 일러스트, 유닛 이름, 성급(별 아이콘)
  - **스탯 목록 영역**: 14종 스탯을 행(Row)으로 나열 — 각 행은 `아이콘 | 스탯명 | 최종값`

## 3. TACD-192: UI 제작

### StatInfoPanel 프리팹

- Screen Space Overlay Canvas 위에 배치
- 레이아웃:
  - **Panel (배경)** — Image 컴포넌트, 반투명 배경
    - **Header** — 수평 배치
      - **NameText** — 유닛 이름 (TextMeshProUGUI)
      - **StarContainer** — 성급 별 아이콘, 이름 오른쪽에 나열 (HorizontalLayoutGroup)
      - **CloseButton** — 오른쪽 끝 X 버튼 (Button + Image)
    - **UnitInfoArea** — 수평 배치
      - **IllustrationImage** — 유닛 일러스트 (Image)
    - **Separator** — 구분선 (Image)
    - **StatListArea** — 스탯 행 목록 (VerticalLayoutGroup + ContentSizeFitter)
      - StatRow × 14 (동적 생성)

### StatRow 프리팹

- 수평 배치 (HorizontalLayoutGroup)
  - **IconImage** — 스탯 아이콘 (Image)
  - **StatNameText** — 스탯명 (TextMeshProUGUI)
  - **StatValueText** — 최종값 (TextMeshProUGUI)

### CloseButton 동작

- Button.onClick에서 `StatInfoPanel.Hide()` 호출

## 5. TACD-194: 스탯 연결

### StatInfoPanel.cs — 스탯 목록 갱신

- `Show(Unit unit)` 호출 시:
  - `unit.StatSheet.Enumerate()`로 14종 `(UnitStatKind, UnitStat)` 순회
  - 각 항목마다 StatRow 프리팹을 Instantiate (또는 풀링된 행 재사용)
  - StatRow에 데이터 바인딩

### StatRow.cs

- `Bind(UnitStatKind kind, UnitStat stat)`:
  - 스탯명: `UnitStatKind`의 `InspectorName` 어트리뷰트에서 한글명 추출
  - 최종값: `stat.CurrentValue` 표시
  - 아이콘: 스탯 종류별 아이콘 (스탯 아이콘 에셋이 없으면 임시 placeholder)

### InspectorName 추출 유틸리티

- `UnitStatKind` enum의 `[InspectorName]` 어트리뷰트에서 한글 표기명을 리플렉션으로 가져오는 헬퍼 메서드 작성
- 위치: `UnitStatsByLevelData.cs` 또는 별도 유틸 클래스

### 데이터 흐름

```
Unit.StatSheet.Enumerate()
└── foreach (UnitStatKind kind, UnitStat stat)
    └── StatRow.Bind(kind, stat)
        ├── kind → InspectorName → StatNameText.text
        ├── stat.CurrentValue → StatValueText.text
        └── kind → 아이콘 매핑 → IconImage.sprite (placeholder)
```

## 6. TACD-196: 유닛 선택/해제로 패널 열기·닫기

### 이벤트 정의

```
OnUnitSelectedEventDto : IGameEvent
└── Unit unit  // 선택된 유닛

OnUnitDeselectedEventDto : IGameEvent
└── (필드 없음)
```

### UnitSelector.cs

- 씬에 하나 배치 (MonoBehaviour)
- `Update()`에서 마우스 클릭(또는 터치) 감지
- `Physics2D.Raycast`로 클릭 위치의 유닛 감지
  - 유닛 히트 → `GlobalEventBus.Publish(new OnUnitSelectedEventDto(unit))`
  - 빈 곳 히트 또는 UI 클릭 → 무시 (패널은 X 버튼으로만 닫음)

### StatInfoPanel.cs — 이벤트 구독

- `OnEnable`: `GlobalEventBus.Subscribe<OnUnitSelectedEventDto>(OnUnitSelected)`
- `OnDisable`: `GlobalEventBus.Unsubscribe<OnUnitSelectedEventDto>(OnUnitSelected)`
- `OnUnitSelected` → `Show(evt.unit)`
- CloseButton 클릭 → `Hide()`

### 패널 열기/닫기 흐름

```
유닛 클릭
→ UnitSelector: Physics2D.Raycast로 Unit 감지
→ GlobalEventBus.Publish(OnUnitSelectedEventDto)
→ StatInfoPanel.OnUnitSelected: Show(unit)
→ 유닛 정보 + 스탯 바인딩 + 패널 활성화

X 버튼 클릭
→ StatInfoPanel.Hide()
→ 패널 비활성화
```
