## 3. TACD-192: UI 제작

### StatInfoPanel 프리팹

- Screen Space Overlay Canvas 위에 배치
- 레이아웃:
  - **Panel (배경)** — Image 컴포넌트, 반투명 배경
    - **Header** — 수평 배치
      - **NameText** — 유닛 이름 (TextMeshProUGUI)
      - **StarContainer** — 성급 별 아이콘, 이름 오른쪽에 나열 (HorizontalLayoutGroup)
      - **CloseButton** — 오른쪽 끝 X 버튼 (Button + Image)
    - **UnitInfoArea** — 수평 배치
      - **IllustrationImage** — 유닛 일러스트 (Image)
    - **Separator** — 구분선 (Image)
    - **StatGrid** — 주요 스탯 목록 (GridLayoutGroup, 2열)
      - StatCell × 8 (동적 생성: MaxHealth, PhysicalAttack, PhysicalDefense, MagicAttack, MagicDefense, AttackSpeed, CooldownReduction, AttackRange)
    - **Separator** — 구분선 (Image)
    - **SubStatGrid** — 부가 스탯 목록 (GridLayoutGroup, 2열)
      - StatCell × 6 (동적 생성: MoveSpeed, CriticalChance, CriticalDamageMultiplier, StatusResistance, DamageDealtIncrease, DamageReduction)

### StatCell 프리팹

- 수평 배치
  - **StatLabel** — 스탯명 (TextMeshProUGUI)
  - **StatValue** — 최종값 (TextMeshProUGUI)

### CloseButton 동작

- Button.onClick에서 `StatInfoPanel.Hide()` 호출

## 5. TACD-194: 스탯 연결

### StatInfoPanel.cs — 스탯 목록 갱신

- `Show(Unit unit)` 호출 시:
  - `MainStats` 배열 (8종)과 `SubStats` 배열 (6종)으로 분리하여 순회
  - 각 항목마다 StatCell 프리팹을 Instantiate → 해당 Grid에 배치
  - StatCell에 데이터 바인딩
- `Hide()` 시 동적 생성된 셀 모두 Destroy

### StatCell.cs

- `Bind(UnitStatKind kind, UnitStat stat)`:
  - 스탯명: `UnitStatKindHelper.GetDisplayName(kind)` — InspectorName 한글명
  - 최종값: `stat.CurrentValue` → FormatValue (퍼센트/소수점 포맷)

### UnitStatKindHelper.cs

- `UnitStatKind` enum의 `[InspectorName]` 어트리뷰트에서 한글 표기명을 리플렉션으로 추출
- 위치: `StatInfoPanel/Scripts/UnitStatKindHelper.cs`

### 데이터 흐름

```
StatInfoPanel.Show(unit)
├── MainStats foreach → SpawnCell(statGrid, kind, stat)
└── SubStats foreach → SpawnCell(subStatGrid, kind, stat)
    └── StatCell.Bind(kind, stat)
        ├── kind → UnitStatKindHelper.GetDisplayName → StatLabel.text
        └── stat.CurrentValue → FormatValue → StatValue.text
```

## 6. TACD-196: 유닛 선택/해제로 패널 열기·닫기

### 이벤트 정의

```
OnUnitSelectedEvent : IGameEvent
└── Unit Unit  // 선택된 유닛 (null이면 선택 해제)
```

### UnitSelector.cs

- 씬에 하나 배치 (MonoBehaviour, Managers 하위)
- `Update()`에서 마우스 왼쪽 클릭 감지 (Input System: `Mouse.current.leftButton.wasPressedThisFrame`)
- `Physics2D.Raycast`로 클릭 위치의 유닛 감지
  - 유닛 히트 → `GlobalEventBus.Publish(new OnUnitSelectedEvent(unit))`
  - 빈 곳 클릭 → `GlobalEventBus.Publish(new OnUnitSelectedEvent(null))` (패널 닫힘)

### StatInfoPanel.cs — 이벤트 구독

- `Awake`: `GlobalEventBus.Subscribe<OnUnitSelectedEvent>(OnUnitSelected)` + `gameObject.SetActive(false)`
- `OnDestroy`: `GlobalEventBus.Unsubscribe<OnUnitSelectedEvent>(OnUnitSelected)`
- `OnUnitSelected` → Unit != null이면 `Show(unit)`, null이면 `Hide()`
- CloseButton 클릭 → `Hide()`
- 주의: OnEnable/OnDisable이 아닌 Awake/OnDestroy 사용 — 패널이 비활성화 상태에서도 이벤트 수신 필요

### 패널 열기/닫기 흐름

```
유닛 클릭
→ UnitSelector: Physics2D.Raycast로 Unit 감지
→ GlobalEventBus.Publish(OnUnitSelectedEvent(unit))
→ StatInfoPanel.OnUnitSelected: Show(unit)
→ 유닛 정보 + 스탯 바인딩 + 패널 활성화

빈 곳 클릭 또는 X 버튼 클릭
→ GlobalEventBus.Publish(OnUnitSelectedEvent(null)) 또는 CloseButton.onClick
→ StatInfoPanel.Hide()
→ 동적 셀 Destroy + 패널 비활성화
```
