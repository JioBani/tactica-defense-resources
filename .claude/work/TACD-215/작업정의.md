# TACD-215 작업정의

## 일감 요약
- **제목**: HP 0 → 다운/비활성화, 침략자 무시 통과
- **완료 기준**:
  - HP 0 도달 시 다운 상태로 전이하고 비활성화된다
  - 침략자가 다운된 유닛을 무시하고 통과한다

## 현재 구현 상태

### 이미 구현됨 (5건)

| 기능 | 위치 | 상태 |
|------|------|------|
| HP 0 → Downed 전이 | `ActionStateController.CheckStateTransition()` L21-25 | 완료 |
| 공격 중지 | `Attacker.OnStateExit(Attack)` → DynamicRepeater 정지 | 완료 |
| 이동 중지 | `Mover.OnStateExit(Move)` → velocity = 0 | 완료 |
| 침략자가 Downed 타겟 무시 | `Attacker.TryAcquireTarget()` L114, `OnStateRun` L147 | 완료 |
| 침략자 Downed 시 디스폰 | `Aggressor.OnStateEnter(Downed)` → DeSpawn | 완료 |

### 미구현: 스킬 실행 미차단 (1건)

- **SkillCaster.cs** — `IStateListener<PhaseType>`만 구현, `ActionStateType`은 미감시
- `Update()`에서 `_isSkillReady && _skill.CanCast()` 체크만 수행
- 유닛 Downed 후에도 `_victim` 참조가 남아있으면 스킬 시전 가능 (버그)
- **수정 방안**: SkillCaster에 `IStateListener<ActionStateType>` 추가하여 Downed 진입 시 `_isSkillReady = false` + 타이머 정지

## 작업 내용

SkillCaster에 ActionStateType Downed 감시 로직을 추가하여 다운 시 스킬을 중지시킨다.
