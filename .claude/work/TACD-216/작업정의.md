# TACD-216 작업정의

## 일감 요약
- **제목**: 유닛의 HP 변화가 체력바에 실시간 반영된다
- **완료 기준**: HP 변화 시 체력바가 즉시 갱신된다

## 현재 구현 상태: 이미 완료됨

### 연결 구조 (구현 완료)

```
UnitStatSheet.Health setter
  → OnHealthChange 이벤트 발화
    → Unit.cs에서 구독: healthBar.Display(value / MaxHealth)
      → HealthBar.Display(rate): insideSprite 크기를 rate 비율로 조절
```

| 기능 | 위치 | 상태 |
|------|------|------|
| Health 변경 시 이벤트 발화 | `UnitStatSheet.cs` L39 | 완료 |
| 이벤트 구독 → HealthBar 갱신 | `Unit.cs` L38-40 | 완료 |
| 체력바 비율 표시 | `HealthBar.cs` L10-19 | 완료 |

## 작업 내용

이미 구현 완료되었으므로 추가 작업 없이 완료 처리한다.
