# TACD-214 작업정의

## 일감 요약
- **제목**: 유닛이 데미지를 받으면 HP가 감소한다
- **완료 기준**: 유닛이 데미지를 받으면 HP가 정상적으로 감소한다

## 현재 구현 상태: 이미 완료됨

### 데미지 파이프라인 (구현 완료)

```
Attacker → AttackCast → RangeAttackExecutor → DamageCalculator.Calculate() → Victim.Hit(damage) → UnitStatSheet.Health -= damage
```

1. **Victim.Hit()** (`Victim.cs`): `unit.StatSheet.Health -= damage` — 데미지를 HP에서 직접 차감
2. **UnitStatSheet.Health setter** (`UnitStatSheet.cs`): `Mathf.Clamp(value, 0, MaxHealth)` — HP가 0 미만으로 내려가지 않음
3. **DamageCalculator.Calculate()** (`DamageCalculator.cs`): 기획서 1.3 데미지 공식 완전 구현
   - 물리/마법 공격력 → 치명타 → 방어력 → 증폭 → 감소 → 내림

### 기존 테스트 (이미 존재)

- `DamageCalculatorTests.cs` — 데미지 공식 11개 테스트 케이스
- `UnitStatSheetTests.cs` — Health 클램핑 및 이벤트 발화 테스트

## 작업 내용

이미 구현과 테스트가 완료되었으므로 추가 작업 없이 완료 처리한다.
