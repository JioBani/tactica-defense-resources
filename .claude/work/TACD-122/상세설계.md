## 1. 전체 구조 개요

```
UnitStatsByLevelData (SO, 별 등급별 기본값)
  → UnitStatSheet.Init()
    → 각 UnitStat.SetBaseValue()

외부 효과 (버프/시너지/전술지원)
  → StatModifier 생성
    → UnitStat.AddModifier() / RemoveModifiersBySource()
      → Recalculate() → OnChange 이벤트

Attacker/Mover 등 컴포넌트
  → UnitStat.OnChange 구독 → 동적 반영

데미지 발생
  → DamageCalculator.Calculate(공격자 StatSheet, 피격자 StatSheet, ...)
    → Victim.Hit(최종 데미지)
```

## 3. TACD-185: 유닛 능력치 모듈

### 3.1 UnitStat 클래스 (기존 UnitStats\<T\> 대체)

**파일:** `Assets/Scenes/Battle/Feature/Unit/Unit/UnitStats/UnitStats.cs` (파일명 유지, 클래스명 변경)

기존 `UnitStats<T>`를 `UnitStat`으로 변경한다. float 전용으로 단순화하고 수정자를 지원한다.

```csharp
namespace Scenes.Battle.Feature.Units.UnitStats
{
    public class UnitStat
    {
        private readonly StatCalculationMode _mode;
        private float _baseValue;
        private float _currentValue;
        private readonly List<StatModifier> _modifiers = new();

        public event Action<float> OnChange;

        public UnitStat(StatCalculationMode mode = StatCalculationMode.Additive)
        {
            _mode = mode;
        }

        /// 기본값 (ScriptableObject 기반, 별 등급에 따라 결정)
        public float BaseValue => _baseValue;

        /// 최종값 (기본 + 수정자 반영). 외부에서는 이 값을 사용한다.
        public float CurrentValue => _currentValue;

        /// 기본값 설정 (Init, 별 등급 변경 시)
        public void SetBaseValue(float value)
        {
            _baseValue = value;
            Recalculate();
        }

        /// 수정자 추가 → 자동 재계산
        public void AddModifier(StatModifier modifier) { ... }

        /// 수정자 제거 → 자동 재계산
        public void RemoveModifier(StatModifier modifier) { ... }

        /// 특정 출처의 수정자 일괄 제거 → 자동 재계산
        public void RemoveModifiersBySource(object source) { ... }

        /// 모든 수정자 제거 → 자동 재계산
        public void ClearModifiers() { ... }

        private void Recalculate()
        {
            float newValue = _mode switch
            {
                StatCalculationMode.Additive => CalculateAdditive(),
                StatCalculationMode.SeparatedMultiplicative => CalculateSeparatedMultiplicative(),
            };

            if (Mathf.Approximately(_currentValue, newValue)) return;
            _currentValue = newValue;
            OnChange?.Invoke(_currentValue);
        }
    }
}
```

### 3.2 계산 공식

#### Additive (일반 스탯)

대부분의 스탯(공격력, 체력, 공격속도 등)에 적용.

```
flatSum     = 모든 Flat 수정자의 Value 합산
percentSum  = 모든 Percent 수정자의 Value 합산
최종값       = BaseValue × (1 + percentSum) + flatSum
```

```csharp
private float CalculateAdditive()
{
    float flatSum = 0f;
    float percentSum = 0f;

    foreach (var mod in _modifiers)
    {
        if (mod.Type == StatModifierType.Flat) flatSum += mod.Value;
        else percentSum += mod.Value;
    }

    return _baseValue * (1f + percentSum) + flatSum;
}
```

#### SeparatedMultiplicative (증감 분리 곱연산)

물리 방어력, 마법 방어력에 적용. 기획서 1.2.3 규칙.

```
증가 수정자: values > 0 → 곱연산  → increaseResult = 1 - ∏(1 - vi)
감소 수정자: values < 0 → 곱연산  → decreaseResult = 1 - ∏(1 - |vi|)
최종값 = BaseValue + increaseResult - decreaseResult
```

BaseValue는 별도로 더한다 (기본 방어력).

```csharp
private float CalculateSeparatedMultiplicative()
{
    float increaseProduct = 1f;
    float decreaseProduct = 1f;

    foreach (var mod in _modifiers)
    {
        if (mod.Value > 0)
            increaseProduct *= (1f - mod.Value);
        else if (mod.Value < 0)
            decreaseProduct *= (1f - Mathf.Abs(mod.Value));
    }

    float increaseResult = 1f - increaseProduct;
    float decreaseResult = 1f - decreaseProduct;

    return _baseValue + increaseResult - decreaseResult;
}
```

### 3.4 UnitStatSheet 수정

**파일:** `Assets/Scenes/Battle/Feature/Unit/Unit/UnitStats/UnitStatSheet.cs`

#### 변경점

1. `UnitStats<float>` → `UnitStat` 으로 변경
2. `DamageReduction` 스탯 추가
3. 방어력 스탯에 `SeparatedMultiplicative` 모드 적용
4. Health를 스탯에서 분리 → 별도 프로퍼티로 관리

```csharp
public class UnitStatSheet
{
    private UnitStatsByLevelData _data;

    // ── 14종 능력치 (수정자 지원) ──
    public readonly UnitStat MaxHealth = new();
    public readonly UnitStat PhysicalAttack = new();
    public readonly UnitStat MagicAttack = new();
    public readonly UnitStat PhysicalDefense = new(StatCalculationMode.SeparatedMultiplicative);
    public readonly UnitStat MagicDefense = new(StatCalculationMode.SeparatedMultiplicative);
    public readonly UnitStat AttackSpeed = new();
    public readonly UnitStat AttackRange = new();
    public readonly UnitStat MoveSpeed = new();
    public readonly UnitStat CriticalChance = new();
    public readonly UnitStat CriticalDamageMultiplier = new();
    public readonly UnitStat CooldownReduction = new();
    public readonly UnitStat StatusResistance = new();
    public readonly UnitStat DamageDealtIncrease = new();
    public readonly UnitStat DamageReduction = new();

    // ── 현재 체력 (런타임 값, 수정자 대상 아님) ──
    private float _health;
    public float Health
    {
        get => _health;
        set
        {
            float clamped = Mathf.Clamp(value, 0f, MaxHealth.CurrentValue);
            if (Mathf.Approximately(_health, clamped)) return;
            _health = clamped;
            OnHealthChange?.Invoke(_health);
        }
    }
    public event Action<float> OnHealthChange;

    public void Init(UnitStatsByLevelData data, int star = 1)
    {
        _data = data;

        // 모든 스탯의 수정자 초기화 + 기본값 설정
        foreach (var (kind, stat) in Enumerate())
        {
            stat.ClearModifiers();
            stat.SetBaseValue(data.GetStat(kind, star));
        }

        // 현재 체력 = 최대 체력
        Health = MaxHealth.CurrentValue;

        // MaxHealth 변경 시 현재 체력 상한 보정
        MaxHealth.OnChange += newMax =>
        {
            if (_health > newMax) Health = newMax;
        };
    }

    public UnitStat Get(UnitStatKind kind) => kind switch { ... };
    public IEnumerable<(UnitStatKind kind, UnitStat stat)> Enumerate() { ... }
}
```

```
효과 발생 (버프/시너지/전술지원)
  → new StatModifier(source, type, value)
  → UnitStat.AddModifier(modifier)
    → Recalculate() → OnChange

효과 해제
  → UnitStat.RemoveModifiersBySource(source)
```

### 6.2 DamageCalculator (static)

**파일 (신규):** `Assets/Scenes/Battle/Feature/Unit/Damage/DamageCalculator.cs`

기획서 1.3의 공식을 그대로 구현한다.

```csharp
namespace Scenes.Battle.Feature.Units.Damage
{
    public static class DamageCalculator
    {
        /// <summary>
        /// 기획서 1.3 데미지 계산 공식.
        /// 순서: 기본 데미지 → 치명타 → 방어력 → 피해 증가 → 피해 감소 → 소수점 버림(최소 0)
        /// </summary>
        public static float Calculate(
            UnitStatSheet attackerStats,
            UnitStatSheet victimStats,
            DamageType damageType,
            float skillCoefficient = 1f)
        {
            // 1. 기본 데미지
            float baseDamage = damageType == DamageType.Physical
                ? attackerStats.PhysicalAttack.CurrentValue
                : attackerStats.MagicAttack.CurrentValue;

            baseDamage *= skillCoefficient;

            // 2. 치명타
            if (Random.value < attackerStats.CriticalChance.CurrentValue)
                baseDamage *= attackerStats.CriticalDamageMultiplier.CurrentValue;

            // 3. 방어력 적용
            float defense = damageType == DamageType.Physical
                ? victimStats.PhysicalDefense.CurrentValue
                : victimStats.MagicDefense.CurrentValue;
            baseDamage *= (1f - defense);

            // 4. 피해 증가 (공격자)
            baseDamage *= (1f + attackerStats.DamageDealtIncrease.CurrentValue);

            // 5. 피해 감소 (피격자)
            baseDamage *= (1f - victimStats.DamageReduction.CurrentValue);

            // 6. 소수점 버림, 최소 0
            return Mathf.Max(0f, Mathf.Floor(baseDamage));
        }
    }
}
```

## 10. 데이터 흐름 요약

```
[유닛 스폰]
UnitLoadOutData.Stats (SO)
  → UnitStatSheet.Init(data, star)
    → 각 UnitStat.SetBaseValue()
    → Health = MaxHealth.CurrentValue

[버프/시너지 적용]
new StatModifier(source, type, value)
  → UnitStat.AddModifier()
    → Recalculate() → OnChange
      → Attacker: 사거리/공속 갱신
      → Mover: 이동속도 갱신

[공격 발생]
Attacker → AttackCast.Cast() → Projectile → RangeAttackExecutor.Execute()
  → DamageCalculator.Calculate(공격자 StatSheet, 피격자 StatSheet, Physical)
    → 기본 데미지 × 치명타 × 방어력 × 피해증가 × 피해감소
  → Victim.Hit(최종 데미지)
    → StatSheet.Health -= damage
      → OnHealthChange → HealthBar.Display()

[버프/시너지 해제]
UnitStat.RemoveModifiersBySource(source)
  → Recalculate() → OnChange → 컴포넌트 자동 갱신
```
