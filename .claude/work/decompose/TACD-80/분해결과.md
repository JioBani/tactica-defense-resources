```
유닛 공통 라이프사이클
│
├─ 행동 상태 전이 (유닛의 생존 흐름)
│   ├─ 💻 대기 (Idle) — 적 미감지 시 대기
│   ├─ 💻 공격 상태 진입 — 적 감지 시 전이

│
├─ 공격 (적에게 데미지를 가하는 동작)
│   ├─ ❓ 타겟 선택 (타겟팅 규칙 — 기획 누락)
│   ├─ ❓ 사거리 판정 (사거리 규칙 — 기획 누락)

│   └─ 💻 쿨타임 관리
```
