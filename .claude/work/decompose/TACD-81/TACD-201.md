# TACD-201: [전장 - 소환수] - 합성 (승급)

> 상태: 완료 (8단계)

---

## 5단계: Sub-task 도출 ✅

### 5-1. 동작 트리 상세화

```
합성 (승급)
├─ 기본 합성 (1성→2성, 2성→3성)
│   ├─ 💻 동일 소환수(같은 종류 + 같은 성급) 3개 보유 감지
│   ├─ 💻 대기석 + 전장 모든 위치의 소환수를 합성 대상에 포함
│   ├─ 💻 정비 페이즈에서만 합성 발생 (전투 중 불가)
│   ├─ 💻 재료 3개 소모 → 상위 성급 소환수 생성
│   ├─ 💻 결과 소환수는 가장 왼쪽 위 재료 위치에 생성
│   └─ 💻 연쇄 합성 처리 (결과로 다시 3개 충족 시)
├─ 강화 합성 (3성 이후)
│   ├─ 💻 3성 + 동일 종류 2성 1개 → 3성+1강
│   └─ 💻 강화 단계 반복 (+1강 + 2성 → +2강 ...)
├─ 승급 효과
│   ├─ 💻 성급에 따른 기본 스탯 변화 적용
│   └─ 💻 경우에 따라 스킬 강화 적용
└─ 합성 시 상태 처리
    └─ 💻 기존 버프 상태 유지
```

### 5-2. 보조 관점 체크

**데이터/속성:**
- 성급별 기본 스탯 테이블 → 스케일링 디자인 영역 (수치), 구현 영역 (데이터 구조)
- 강화 단계별 스탯 증가 테이블 → 동일

**UI/정보 표시:**
- 성급 표시 → 구현 영역 (UI), 비주얼 영역 (디자인)
- 합성 가능 표시 → 표시하지 않음 (결정 완료)
- 합성 발생 피드백/연출 → 비주얼 영역

**기반 작업:**
- 소환수 인스턴스에 성급 속성 → TACD-202 (라이프사이클) 영역
- 소환수 보유 목록 관리 → TACD-202 (라이프사이클) 영역
