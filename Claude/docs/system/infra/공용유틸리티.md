# 공용유틸리티 — 시스템 관점

> 이 문서는 AI 에이전트가 코드 레이어에 맞게 이해하고 빠르게 코드를 찾아가도록 하는 시스템 관점 컨텍스트 문서다.
> 코드가 단일 진실이다. 이 문서는 코드를 재설명하지 않는다 — "인덱스 + α"에 한정한다.
>
> - **분류**: 인프라 — UR 미매핑 횡단 기반. 특정 사용자 기능에 묶이지 않는 범용 공용 유틸리티/속성/직렬화/인스펙터 보조의 **묶음 인덱스**. 각각은 별도 문서로 만들 가치는 낮으나 누락 없이 색인하기 위해 한 문서로 모은다.

## 1. 한 줄 정의

여러 기능이 공통으로 올라타는 작은 횡단 유틸리티(안전 순회 컬렉션·인스펙터 직렬화/속성·트윈/순회 확장·UI 보조·공용 enum)를 한 곳에서 찾도록 색인하는 인프라 묶음.

## 2. 주요 추상 매핑

각 유틸을 진입점 식별자 → 위치 → 한 줄 용도로 색인한다. 도메인 개념이 거의 없는 순수 기술 구성요소 묶음이므로 "주요 추상" 으로 읽는다.

| 주요 추상 | 코드 식별자 | 위치(파일 경로) | 용도 (한 줄) |
|----------|------------|----------------|-------------|
| 안전 순회 리스트 | `SafeIterationList<T>` | `Assets/Common/Scripts/SafeIterationList/SafeIterationList.cs` | 순회 중 Add/Remove를 버퍼링해 컬렉션 수정 예외를 막는 리스트 |
| 직렬화 가능 딕셔너리 | `SerializableDictionary<TKey,TValue>` / `SerializableDictionaryBase` | `Assets/Common/Scripts/SerializableDictionary/SerializableDictionary.cs` | 인스펙터에서 편집 가능한 Dictionary 래퍼 |
| 인스펙터용 시간값 | `SerializableTime` (struct) | `Assets/Common/Scripts/SerializableTime/SerializableTime.cs` | hh:mm:ss.fff 입력 ↔ 초/`TimeSpan` 변환 래퍼 |

| 스프라이트 미리보기 속성 | `SpritePreviewAttribute` | `Assets/Common/Scripts/SpritePreview/SpritePreviewAttribute.cs` | Sprite 필드 아래 미리보기를 그리는 `PropertyAttribute` |

## 3. 구조 맵 (위치 중심)

디렉토리 묶음이다. 각 디렉토리가 독립 유틸이며 진입점은 §2·§3.1 참조.

- **순회/컬렉션**: `Assets/Common/Scripts/SafeIterationList/`
- **직렬화 래퍼**: `Assets/Common/Scripts/SerializableDictionary/`(+`Editor/` Drawer), `Assets/Common/Scripts/SerializableTime/`(+`Editor/` Drawer)
- **인스펙터 속성 + Drawer**: `Assets/Common/Scripts/InspectorHint/`(+`Editor/InspectorHintDrawer.cs`), `Assets/Common/Scripts/InspectorDescriptionAttribute/`(+`Editor/UniversalInspectorDescriptionEditor.cs`), `Assets/Common/Scripts/SpritePreview/`(+`Editor/SpritePreviewDrawer.cs`)
- **에디터 툴**: `Assets/Common/Scripts/GridArranger/Editor/GridArrangerWindow.cs`
- **확장 메서드**: `Assets/Common/Scripts/Extensions/DOTweenExtensions.cs`, `Assets/Common/Scripts/TransformChildrenIterator/TransformChildrenIterator.cs`
- **UI 보조**: `Assets/Common/Scripts/Selectable/Selectable2D.cs`, `Assets/Common/Scripts/BubbleMessage/`(`BubbleMessage`·`BubbleMessageSpawner`·`BubbleMessageConfig`·`BubbleMessageParams`)
- **공용 enum**: `Assets/Common/Scripts/Enums/Fraction.cs`

### 3.2 의존 관계

- **밖으로 의존**: `Common.Scripts.GlobalEventBus`(`Selectable2D`가 `OnObjectSelectedEvent` 발행), `Common.Scripts.ObjectPool`·`Common.Scripts.SceneSingleton`(`BubbleMessage`·`BubbleMessageSpawner`), DOTween(`DOTweenExtensions`·`BubbleMessage`), UniTask(`DOTweenExtensions`), TextMeshPro(`BubbleMessage`)
- **밖에서 의존**: 프로젝트 전반의 다수 기능 코드가 공통으로 소비하는 횡단 유틸 묶음 — 단일 대표 소비처 없음

## 4. +α — 코드만 봐선 모르는 것

- **묶음 의도**: 각 유틸은 너무 작아 독립 인프라 문서로 둘 가치가 낮지만, 누락 없이 색인하기 위해 한 문서로 모았다. 나중에 특정 유틸이 커지면 별도 문서로 분리할 수 있다.
- **비자명한 관례**: 인스펙터 속성 3종(`InspectorHint`·`InspectorDescription`·`SpritePreview`)은 런타임 어셈블리의 속성 정의와 `Editor/`의 Drawer/Editor가 짝을 이룬다 — 동작을 바꾸려면 두 곳을 함께 본다.
- **비자명한 제약**: `SafeIterationList`는 중첩 순회 깊이를 추적해 가장 바깥 순회가 끝날 때만 대기 변경을 반영한다(즉시 반영 아님). `GridArrangerWindow`는 에디터 전용(`#if UNITY_EDITOR`)이라 런타임 코드에서 참조하지 않는다.
