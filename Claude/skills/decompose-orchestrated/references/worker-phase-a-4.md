# 워커: Task Jira 등록 (4단계)

이 파일은 서브에이전트가 직접 읽고 수행하는 지시서이다.

## 입력 파라미터

- **에픽 티켓 번호**: 프롬프트에서 전달받음
- **작업 파일**: 분해결과.md 경로

## 사전 읽기

1. **작업 파일** (분해결과.md)을 읽어 3단계 Task 목록을 확인한다.
2. **이슈 작성 요령**: `Document/프로젝트 관리/구현 이슈 작성 요령.md`를 읽는다.
3. **Jira 지침**: `.claude/docs/jira.md`를 읽는다.

## 수행 절차

1. 분해결과.md의 Task 목록을 기반으로, 각 Task를 에픽의 하위 작업으로 Jira에 등록한다.
2. 이슈 작성 요령의 **Task 작성 요령** 섹션을 따른다:
   - **제목**: `[에픽 제목] - {기능명}`
   - **설명**: Task가 다루는 기능의 범위를 간략히 기술
   - **상태**: 하위 이슈 중 기획 누락이 있으면 "기획 필요"로 설정
3. 등록된 Jira 티켓 번호를 작업 파일에 기록한다.

## Jira MCP 도구 사용 시 주의

- description 필드에 `\n` 이스케이프를 사용하지 않는다. 실제 줄바꿈을 사용한다.
- MCP 도구가 markdown을 Jira wiki markup으로 자동 변환한다.

## 작업 파일 기록

분해결과.md에 4단계 결과를 추가한다:

```markdown
## 4단계: Task Jira 등록 ✅

| # | Task | Jira | Sub-task 분해 |
|---|------|------|--------------|
| 1 | {기능명} | TACD-XXX | - |
| 2 | {기능명} | TACD-XXX | - |
```

상태 표시를 업데이트한다:

```markdown
> 현재 Phase: B
> 다음 작업: B-1 (Sub-task 도출 + 교차 검증, 병렬)
```
