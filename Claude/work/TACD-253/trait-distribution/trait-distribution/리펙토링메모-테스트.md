## 후보 식별 시 제외한 항목 (참고)

- **§E-4 매개변수화 (TC-103 / TC-104)**: traitPool null vs 빈 컬렉션 두 케이스가 동일 가드 경로를 트리거하나, 결정 테이블·동치 분할의 별개 entry 라 분리 유지가 가독성에 우월. 컨벤션 §E-4 단서 "통합이 가독성을 떨어뜨리면 통합하지 않는다" 적용.
- **공통 4-summoner 픽스처 추출**: TC-101 / TC-102 / TC-203 이 모두 4 정상 소환술사를 사용하나, 각 테스트의 의도(전체 배정 / 풀 멤버 검증 / Store 통합)가 다르고 인라인 구성이 한 줄이라 추출 이득이 미미. 유지.
- **§T-2 static 함수 제한**: 본 파일의 static 헬퍼들은 정적 상태(SceneSingleton._instance, GlobalEventBus.Handlers) 에 접근하는 테스트 인프라로 인스턴스 함수화 시 의미가 흐려진다. trait-store 리펙토링 사이클에서 합의된 패턴과 동일하게 유지.
