# TACD-247: 침략자 프리뷰 목록 (종류·성급 기준 그룹핑)

> 상위 이슈: TACD-236 [전장 - 침략자] - 웨이브 프리뷰
> 브랜치: TACD-247

---

## 작업 내용

정비 페이즈에서 다음 웨이브에 등장할 침략자를 **종류 + 성급** 기준으로 그룹핑하여 프리뷰 목록 UI에 표시한다.

### 현재 상태

- `RoundInfoViewer`가 이미 기본 프리뷰를 제공하고 있음 (AggressorSample 사용)
- 현재는 `aggressor.Unit.ID`로만 그룹핑 → **성급 구분 없음**
- `SpawnEntry`에 성급(star) 필드가 없음 → 모든 침략자가 1성으로 소환됨
- `AggressorSample`은 숫자(count)만 표시 → **성급 표시 없음**

### 접근 방식: 실제 Aggressor 엔티티 + Freeze 상태

AggressorSample 대신 **실제 Aggressor 엔티티를 Freeze 상태로 소환**하여 프리뷰에 사용한다.

- Aggressor는 이미 스프라이트 + HealthBar(성급 색상)를 가지고 있으므로 별도 성급 UI가 불필요
- Freeze 상태는 전환 없는 고정 상태 → 이동/공격/피격 없이 외형만 유지
- (종류 + 성급) 그룹당 대표 Aggressor 1체를 Freeze로 배치하고, 숫자(count)를 오버레이

### 변경 필요 사항

1. `SpawnEntry`에 `star` 필드 추가 (성급별 침략자 스폰 데이터 지원)
2. 프리뷰용 Aggressor 소환 시 star 값 반영 + Freeze 상태 진입
3. `RoundInfoViewer`의 그룹핑 로직을 `(Unit.ID, star)` 기준으로 변경
4. 그룹별 등장 숫자 오버레이 표시 (기존 AggressorSample의 countText 역할)
5. `RoundAggressorManager`에서 전투 소환 시에도 SpawnEntry의 star 값 반영

### 기획 미정 사항

- ❓ 정렬 기준: "등장 위치 기준(우상단→하단→왼쪽)" 상세 명세 미정
  - → 현재 SpawnEntry 순서(라운드 데이터에 정의된 순서)대로 표시하고, 정렬 기준이 확정되면 추후 반영
