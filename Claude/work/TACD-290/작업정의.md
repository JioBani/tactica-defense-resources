# TACD-290: UnitAnimator 컴포넌트 제작

> 상위 이슈: TACD-287 [전장 - 유닛] 애니메이션 프레임워크 제작

## 요구사항

ActionStateController의 상태 변경에 연동하여 애니메이션을 자동 전환하는 런타임 컴포넌트를 제작한다.

## 배경지식

- 설계 문서: `Claude/work/TACD-287/애니메이션 프레임워크 설계.md`
- IStateListener<ActionStateType> 구현으로 상태 변경 구독
- 소환 시 UnitDefinitionData에서 성급별 OverrideController를 가져와 Animator에 할당
- OnStateEnter에서 animator.Play(state.ToString())으로 클립 전환
- 기존 패턴: Mover, Attacker 등이 동일한 IStateListener 패턴을 사용

## 완료 정의

- UnitAnimator가 IStateListener<ActionStateType>을 구현한다
- Init 메서드로 데이터 기반 OverrideController를 Animator에 할당할 수 있다
- ActionState 변경 시 해당 상태의 애니메이션이 자동으로 전환된다
- RegisterListener/UnregisterListener가 올바르게 처리된다
