# TACD-279: 침략자가 라인 끝에 도달하면 소환술사와 교전이 시작된다

> 상위 이슈: TACD-275 [전장 - 소환술사] - 라이프사이클

## 요구사항

침략자가 라인 끝에 도달하면 소환술사와 교전을 시작한다. 현재는 DefenseLine 콜라이더 접촉 시 즉시 패배하는데, 이를 소환술사 교전으로 변경한다.

## 배경지식

- 현재 흐름: 침략자 → DefenseLine 콜라이더 접촉 → LifeCrystalAttacker → OnRoundLoseEventDto → 즉시 RoundLose
- 변경 흐름: 침략자 → 소환술사 Attacker 사거리 진입 → 자동 교전 (기존 Unit 시스템)
- Attacker는 OnTriggerEnter2D + "Victim" 태그 + fraction 판정으로 적을 감지
- 침략자도 Attacker를 가지므로 소환술사(Victim)를 타겟으로 잡고 Attack 상태 전환 → 이동 중단

## 완료 정의

- 침략자가 라인 끝에 도달했을 때 즉시 패배하지 않고 소환술사와 교전한다
- 소환술사가 침략자를 공격하고, 침략자가 소환술사를 공격한다
- 기존 DefenseLine 즉시 패배 로직이 비활성화된다

## 기존 코드 참조

- `LifeCrystalAttacker.cs`: DefenseLine 충돌 → OnRoundLoseEventDto 발행
- `Aggressor.cs`: OnEnterDefenseLine() → DeSpawn
- `Attacker.cs`: OnTriggerEnter2D → "Victim" 태그 + fraction 판정 → 자동 공격
- `ActionStateTransitionService.cs`: victim 존재 → Attack 상태 전환
