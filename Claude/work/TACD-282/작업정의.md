# TACD-282: 소환술사가 다운되면 전투에서 패배한다

> 상위 이슈: TACD-275 [전장 - 소환술사] - 라이프사이클

## 요구사항

소환술사 1명이라도 HP가 0이 되어 Downed 상태가 되면 전투에서 패배(RoundLose)한다.

## 배경지식

- ActionStateTransitionService가 HP ≤ 0 시 자동으로 Downed 전환
- 기존 즉시 패배(DefenseLine)는 TACD-279에서 제거됨
- RoundLose 전환: `RoundManager.Instance.RequestStateChange(PhaseType.RoundLose)` 또는 `GlobalEventBus.Publish(new OnRoundLoseEventDto())`

## 완료 정의

- 소환술사 1명이라도 Downed 상태가 되면 RoundLose가 트리거된다

## 작업 체크리스트

- [x] 소환술사 Downed 감지 → RoundLose 트리거
