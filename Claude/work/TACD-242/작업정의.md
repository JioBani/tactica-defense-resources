# TACD-242: 소환터미널 > 스캔 - 고성급 등장 확률 UI 표시

## 작업 내용

현재 배치 상한 레벨에서의 2성/3성 소환수 등장 확률을 소환터미널 UI에 표시한다.
플레이어가 레벨업 시 확률 변화를 인지할 수 있어야 한다.

## 완료 정의

- 소환터미널 UI에 현재 레벨의 2성/3성 등장 확률이 표시된다
- 레벨업 시 확률 표시가 자동으로 갱신된다

## 관련 참조

- **기획서**: `Document/기획서/07 게임 시스템 기획서.md` > 3.2 소환(구매)
- **StarProbabilityConfig**: `Assets/Common/Data/Configs/StarProbabilityConfig.cs` - 성급 확률 정의 (TACD-233에서 구현 완료)
- **AppearanceRatesPanel**: `Assets/Scenes/Battle/Feature/Ui/Market/AppearanceRatesPanel.cs` - 기존 코스트별 등장 확률 패널 (참고 패턴)
- **MarketUiManager**: `Assets/Scenes/Battle/Feature/Ui/Market/MarketUiManager.cs` - 소환터미널 UI 관리
- **MarketManager**: `Assets/Scenes/Battle/Feature/Market/Scripts/MarketManager.cs` - Level RxValue 보유

## 현재 구조 분석

- `StarProbabilityConfig`에 `PickStar(level)` 메서드는 있으나, 확률 값을 조회하는 공개 메서드가 없음 (`FindEntry`가 private)
- `MarketManager.Level`은 `RxValue<int>`로 구독 가능
- `MarketUiManager`는 이미 `Level.OnChange`를 구독하여 레벨업 비용을 갱신하고 있음
- `AppearanceRatesPanel`은 코스트별 확률만 표시하며, 성급 확률은 미표시
