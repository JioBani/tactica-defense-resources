# TACD-278: 전장 시작 시 소환술사 4명이 라인 끝에 자동 생성/배치된다

> 상위 이슈: TACD-274 [전장 - 소환술사] - 전장 배치

## 요구사항

전장(전투 페이즈) 시작 시, 출전 편성에서 선택한 소환술사 4명을 각 라인 끝(왼쪽)에 자동으로 생성하고 배치한다. 소환술사는 출전 편성 순서대로 위에서부터 배치된다.

## 배경지식

- **소환술사**: 라인 끝에 고정 배치되는 유닛. 침략자가 라인 끝까지 도달하면 소환술사와 교전하며, 소환술사 1명이라도 다운되면 전투 패배.
- **출전 편성**: 전장 입장 전 보유 소환술사 중 4명을 선택하는 시스템. 편성 순서가 라인 배치 순서를 결정한다.
- **라인**: 전장에 4개의 라인이 존재하며, 침략자는 오른쪽에서 왼쪽으로 이동. 소환술사는 라인 왼쪽 끝에 위치.
- **Summoner 프리팹**: TACD-277에서 이미 제작 완료. Summoner 컴포넌트, SkillCaster, Attacker, Victim 등이 구성됨.

## 완료 정의

- 전투 페이즈 시작 시 소환술사 4명이 각 라인 끝에 자동 생성된다
- 출전 편성 순서대로 위 라인(1번)부터 아래 라인(4번)까지 배치된다
- 소환술사가 유닛으로서 정상 동작한다 (Attacker, Victim 등 컴포넌트 활성)
- 전투 종료 시 소환술사가 정리(제거)된다
