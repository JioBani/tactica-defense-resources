# TACD-228: 라운드 종료 시 스킬 쿨다운 초기화

## 작업 내용

라운드(전투 페이즈) 종료 시 모든 소환수의 스킬 쿨다운을 초기화하여, 다음 라운드 시작 시 스킬을 즉시 사용할 수 있게 한다.

## 완료 정의

- 전투 종료 후 모든 소환수의 스킬 쿨다운이 리셋된다
- 다음 라운드 전투 시작 시 스킬을 즉시 사용할 수 있다
- 다운된 소환수도 라운드 종료 시 쿨다운이 초기화된다

## 기획서 참조

- **07 게임 시스템 기획서 > 2.1.1 라운드 간 상태 관리**
  - "스킬 쿨다운은 라운드 종료 시 초기화된다."
  - "다운된 소환수도 라운드 종료 시 HP 전부 회복 상태로 복귀한다."

## 현재 코드 분석

### SkillCaster (`Assets/Scenes/Battle/Feature/Unit/Casting/Caster/SkillCaster.cs`)
- `Timer` 기반으로 쿨다운을 관리
- `OnEnterCombat()`: `_isSkillReady = true`, `_skillTimer.Start()` → 전투 시작 시 스킬 즉시 사용 가능
- `OnExitCombat()`: `_skillTimer.Stop()` → 타이머 정지만 하고 **쿨다운 초기화를 하지 않음**
- Downed 상태 시 `_isSkillReady = false`, `_skillTimer.Stop()` → 스킬 차단

### 문제점
- `OnExitCombat()`에서 `_isSkillReady`를 `true`로 설정하지 않음
- 현재는 `OnEnterCombat()`에서 `_isSkillReady = true`로 설정하여 다음 전투 시작 시에야 스킬이 준비됨
- 라운드 "종료 시" 초기화가 아닌, 다음 전투 "시작 시" 초기화되는 구조

### 참고 패턴: HP 회복 (Defender.cs)
- `ActionStateType.Waiting` 상태 진입 시 `StatSheet.RecoverFullHealth()` 호출
- Waiting 상태는 Maintenance 페이즈 진입 시 전환됨
