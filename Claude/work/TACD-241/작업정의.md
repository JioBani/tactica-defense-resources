# TACD-241: 마켓 슬롯에 소환수 성급 및 마나 표시

## 작업 내용

스캔 결과로 2성 또는 3성 소환수가 등장했을 때, 마켓 슬롯 UI에 성급과 마나 소비량을 시각적으로 표시한다.

## 완료 정의

- 마켓 슬롯에 해당 소환수의 성급(1성/2성/3성)이 표시된다
- 마켓 슬롯에 해당 소환수의 성급에 따른 소비 마나가 표시된다 (1성 1마나, 2성 3마나, 3성 9마나)
- 1성과 고성급 소환수가 시각적으로 구분된다

## 기획서 참조

- `07 게임 시스템 기획서` > 3.2 소환(구매):
  - 1성: 1마나, 2성: 3마나, 3성: 9마나

## 기존 코드 분석

### 현재 구조

1. **DefenderSlot.cs** (`Assets/Scenes/Battle/Feature/Ui/Market/DefenderSlot.cs`): 마켓 슬롯 UI 컴포넌트
   - `MarketDefenderSlot`을 받아 일러스트(`Image`)를 표시
   - 구매 시 일러스트 숨기고 `purchasedText` 표시
   - 성급/마나 표시 UI 요소 없음

2. **MarketDefenderSlot.cs**: 슬롯 데이터 클래스
   - `Star` 속성 존재 (TACD-233에서 추가됨)
   - `UnitLoadOutData` 속성으로 유닛 정보 접근 가능

3. **씬 슬롯 구조** (`Canvas/Market/Market/UnitButtons/DefenderSlot/`):
   ```
   DefenderSlot (Button, DefenderSlot)
   └── Panel (RectMask2D)
       ├── Backgound (Image)
       ├── Illustration (Image, AspectRatioFitter)
       └── PurchasedText (TMP, 비활성)
   ```

### 변경 대상

- `DefenderSlot.cs` - 성급/마나 텍스트 UI 바인딩 및 표시 로직 추가
- 씬 내 DefenderSlot 4개 - 성급 텍스트, 마나 텍스트 UI 요소 추가
