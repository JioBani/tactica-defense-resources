# TACD-227: 라운드 종료 시 모든 소환수 HP 전부 회복 + 복귀

## 작업 내용

라운드(전투 페이즈) 종료 후 Maintenance 진입 시 모든 소환수의 HP를 전부 회복하고,
다운된 소환수도 활성 상태로 복귀시킨다.

## 완료 정의

- 전투 종료 후 생존 소환수의 HP가 최대치로 회복된다
- 다운된 소환수가 HP 전부 회복 상태로 복귀하여 다음 라운드에 참여할 수 있다
- 라운드 승리(End → Maintenance)와 라운드 실패(RoundLose → Maintenance) 모두 동일하게 동작한다

## 설계

### 상태 역할 정리

| 상태 | 시점 | 의미 |
|------|------|------|
| Freeze | 전투 종료 → Maintenance 직전 | 정지. 다운 외형/HP바 유지 |
| **Waiting** (신규) | Maintenance 중 | 회복 완료. 무적. 다음 전투 대기 |

### 전투 종료 → 복귀 흐름

```
Combat → HP 0 → Downed → 전투 종료 → Freeze (다운 외형 유지)
→ Maintenance 진입 → Waiting (ActionStateController가 Phase 감지)
→ Defender가 Waiting 진입 감지 → HP 회복
→ 다음 Combat 시작 → Idle/Move
```

### Downed 전환 조건 제한

기존: HP <= 0이면 어떤 상태에서든 Downed 전환
변경: HP <= 0이면서 **Idle, Move, Attack** 상태일 때만 Downed 전환
- Freeze, Waiting, Downed 상태에서는 Downed 전환하지 않음
