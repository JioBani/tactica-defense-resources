# TACD-260: 테스트용 임시 시너지 효과 구현

> 상위 이슈: TACD-251 [전장 - 시너지] - 시너지 계산 및 효과 적용

## 요구사항

TACD-262에서 구축한 SSE(SynergyStatusEffect) 프레임워크를 기반으로, Jira에 정의된 5종의 시너지 효과를 구현한다.
프레임워크가 실제 게임플레이에서 동작하는지 검증하기 위한 테스트용 구현이다.

### 구현 대상 시너지

**1. 난동꾼** — 난동꾼이 체력을 얻는다.
- (2) 체력 25% / (4) 체력 40% / (6) 체력 60%

**2. 비전 마법사** — 아군이 주문력을 얻는다. 비전 마법사는 더 얻는다.
- (2) 주문력 15, 비전 마법사 25 / (4) 주문력 20, 비전 마법사 50 / (6) 주문력 35, 비전 마법사 70

**3. 프렐요드** — 프렐요드 챔피언이 공격 시 대상을 3초 동안 둔화한다.
- (2) 20% / (4) 35% / (6) 60%

**4. 전쟁기계** — 전쟁기계가 내구력(피해감소)을 얻는다. 체력 50% 이상이면 수치 증가. 전쟁기계 사망 시 다른 전쟁기계가 최대 체력 5% 회복.
- (2) 18% 또는 25% / (4) 20% 또는 30% / (6) 26% 또는 36%

**5. 총잡이** — 총잡이가 공격력을 얻으며, 기본 공격 4회마다 추가 물리 피해를 입힌다.
- (2) 공격력 22%, 100 추가 피해 / (4) 공격력 40%, 200 추가 피해

## 배경지식

### 시너지 프레임워크
- `SynergyController`: Template Method 패턴. SSE 부여를 보장하고 자식이 추가 효과를 확장
- `SynergyControllerFactory`: SceneSingleton. SynergyId별 Controller 생성 및 의존성(SerializeField) 주입
- `SynergyStatusEffect<TContext>`: SSE 추상 클래스. OnSynergyActivated/Deactivated/TierChanged 콜백
- `TierLinkedStatusEffect<TContext>`: 비보유 유닛용 부수 SE. SSE와 동일한 티어 구독 생명주기
- `StatusEffect` 기본 클래스는 `StatusEffectDefinitionData`를 생성자로 주입받음
- SSE/TLSE는 ActiveTier(RxValue)를 구독하여 티어 변경/비활성화를 자체 처리
- 상세 사용법: `Assets/Scenes/Battle/Feature/Synergy/Scripts/README.md` 참조

### 스탯 수정자 시스템
- `UnitStat.AddModifier(StatModifier)` / `RemoveModifiersBySource(source)`
- `StatModifier(source, type, value)` — Flat(고정값) 또는 Percent(%)
- SSE 자신을 source로 사용하면 OnRemove 시 일괄 해제 가능

### 시너지 효과 복잡도 분류
- **스탯 버프 (단순)**: 난동꾼, 총잡이(공격력 부분) — 수정자만 추가/제거
- **차등 버프**: 비전 마법사 — 시너지 보유 여부에 따라 다른 값 적용. "전체 아군 대상" 시너지
- **행동 변경**: 프렐요드(공격 시 둔화), 총잡이(4회 공격 시 추가 피해) — HookProvider/공격 이벤트 구독 필요
- **조건부/반응형**: 전쟁기계 — 체력 조건에 따른 수치 변동, 사망 시 아군 회복

## 완료 정의

- 5종의 시너지 효과가 각각 SSE(+ 필요 시 TLSE)로 구현되어 있다
- SynergyControllerFactory에 5종이 등록되어 있다
- SynergyId에 5종의 시너지 ID가 정의되어 있다
- 스탯 버프 계열 SSE는 활성화 시 수정자를 추가하고, 티어 변경 시 갱신하고, 비활성화 시 제거한다
- 난동꾼: 시너지 보유 유닛의 MaxHealth에 티어별 % 수정자가 적용된다
- 비전 마법사: 전체 아군에게 MagicAttack Flat 수정자가 적용되고, 비전 마법사 보유 유닛은 더 높은 값을 받는다
- 프렐요드: 공격 시 대상에게 3초 둔화(MoveSpeed 감소)가 적용된다
- 전쟁기계: 체력 50% 이상/이하에 따라 DamageReduction 수정자 값이 달라지고, 사망 시 다른 전쟁기계가 최대 체력 5%를 회복한다
- 총잡이: 공격력 % 수정자가 적용되고, 기본 공격 4회마다 추가 물리 피해를 입힌다
- 각 SSE에 대한 유닛 테스트가 존재한다
