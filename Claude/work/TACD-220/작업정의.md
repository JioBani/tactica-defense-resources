# TACD-220: 쿨타임 감소 스탯이 스킬 재사용 대기시간에 반영된다

## 작업 내용

유닛의 "쿨타임 감소(CooldownReduction)" 스탯을 스킬 쿨타임 계산에 반영한다.

### 현재 상태

- `SkillCaster.Initialize()`에서 `SkillDefinitionData.CoolTime`을 그대로 사용
- `UnitStatSheet.CooldownReduction` 스탯은 인프라(enum, 데이터, UI)가 모두 갖춰져 있으나, 실제 쿨타임 계산에 연결되지 않음
- Timer는 `SetMaxTime()`으로 동적 갱신 가능

### 변경 대상

- `SkillCaster.cs`: 쿨타임 계산 시 `CooldownReduction` 스탯을 반영
  - 초기화 시: `effectiveCoolTime = baseCoolTime × (1 - cooldownReduction)`
  - 스탯 변경 시: `UnitStat.OnChange` 구독하여 타이머 MaxTime 동적 갱신

### 계산 공식

```
유효 쿨타임 = 기본 쿨타임 × (1 - 쿨타임 감소율)
```

| 기본 쿨타임 | 감소율 | 유효 쿨타임 |
|------------|--------|------------|
| 5.0초 | 0% | 5.0초 |
| 5.0초 | 20% | 4.0초 |
| 5.0초 | 50% | 2.5초 |

## 완료 정의

- [x] `SkillCaster`가 초기화 시 `CooldownReduction` 스탯을 쿨타임에 반영한다
- [x] 전투 중 `CooldownReduction` 스탯이 변경되면 타이머가 동적으로 갱신된다
- [x] 유닛 테스트로 계산 로직을 검증한다
