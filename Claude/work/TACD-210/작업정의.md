# TACD-210: 소환수 성급 표시

## 작업 내용

소환수의 성급(별 등급)과 강화 단계를 UI에 표시하고, 성급에 따라 HP 바 테두리 색을 변경한다.

## 완료 정의

- 소환수의 성급(1성/2성/3성)이 스탯 UI(StatInfoPanel)에 별 아이콘으로 표시된다
- 소환수의 강화 단계(+1강, +2강 등)가 스탯 UI에 텍스트로 표시된다
- 소환수의 성급에 따라 HP 바의 테두리(outline) 색이 변경된다
  - 1성: 회색
  - 2성: 금색
  - 3성: 주황색
- 합성(승급) 또는 강화 합성 후 표시가 즉시 갱신된다

## 관련 코드

- **UnitStatSheet** (`Assets/Scenes/Battle/Feature/Unit/Unit/UnitStats/UnitStatSheet.cs`)
  - `Star`, `Reinforcement` 속성 보유. 합성 시 UpgradeStar / Reinforce 호출
- **StatInfoPanel** (`Assets/Scenes/Battle/Feature/Ui/StatInfoPanel/Scripts/StatInfoPanel.cs`)
  - `starImages` 배열 존재, 현재 TODO로 하드코딩(1)
  - `UpdateStars(int count)` 메서드로 별 표시
  - 강화 단계 표시 UI 없음 → 추가 필요
- **HealthBar** (`Assets/Scenes/Battle/Feature/Unit/HealthBar/Scripts/HealthBar.cs`)
  - `outlineSprite` SpriteRenderer 보유 → color 변경으로 테두리 색 구현 가능
- **Unit** (`Assets/Scenes/Battle/Feature/Unit/Unit/Scripts/Unit.cs`)
  - `healthBar`, `StatSheet` 보유. StatSheet에서 Star 접근 가능
- **DefenderFusionManager** (`Assets/Scenes/Battle/Feature/Fusion/Scripts/DefenderFusionManager.cs`)
  - 합성 후 `OnDefenderFusedEventDto` 이벤트 발행
