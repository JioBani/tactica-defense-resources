# TACD-283/284: 라운드 종료 시 HP 회복 + 재도전 시 상태 초기화

> 상위 이슈: TACD-275 [전장 - 소환술사] - 라이프사이클

## 요구사항

- 라운드 종료 시 소환술사 HP가 전부 회복된다
- 라운드 실패 후 재도전 시 소환술사 상태가 초기화된다

## 배경지식

- Defender 패턴: Waiting 상태 진입 시 `StatSheet.RecoverFullHealth()` 호출
- Maintenance 페이즈 진입 → ActionState가 Waiting으로 전환 (정상 종료/재도전 모두)
- Summoner에 이미 IStateListener<ActionStateType> 구현됨 (TACD-282)

## 완료 정의

- 라운드 종료 후 Maintenance 진입 시 소환술사 HP가 전부 회복된다
- 재도전 시에도 동일하게 HP가 회복되고 정상 동작한다

## 작업 체크리스트
