# TACD-244: 고성급 소환수 구매 시 성급에 따른 마나 소비 적용

## 작업 내용

현재 `MarketManager.BuyDefender()`는 성급에 관계없이 `UnitDefinitionData.Cost`(1성 기준 가격)만 차감한다.
`UnitLoadOutData.GetCostByStar(int star)` 메서드가 이미 존재하므로, 구매 및 판매 시 성급별 비용을 적용한다.

## 완료 정의

- 소환수 구매 시 `GetCostByStar(star)`로 계산된 성급별 가격이 마나에서 차감된다
- 소환수 판매 시 성급별 가격으로 마나가 환불된다
- 기존 유닛 테스트가 통과한다
- 변경 사항에 대한 유닛 테스트가 추가된다

## 관련 코드

| 파일 | 설명 |
|------|------|
| `Assets/Scenes/Battle/Feature/Market/Scripts/MarketManager.cs` | `BuyDefender()` 라인 166: `slot.UnitLoadOutData.Unit.Cost` → `GetCostByStar(slot.Star)` 필요 |
| `Assets/Scenes/Battle/Feature/Market/Scripts/MarketManager.cs` | `Sell()` 라인 225: `defender.UnitLoadOutData.Unit.Cost` → `GetCostByStar(defender.StatSheet.Star)` 필요 |
| `Assets/Common/Data/Units/UnitLoadOuts/UnitLoadOutData.cs` | `GetCostByStar(int star)` 이미 구현됨 (기본값: `{0, 1, 3, 9}`) |
| `Assets/Scenes/Battle/Feature/Ui/Market/DefenderSlot.cs` | UI 표시는 이미 `GetCostByStar()` 사용 중 — 변경 불필요 |
| `Assets/Scenes/Battle/Feature/Unit/Unit/UnitStats/UnitStatSheet.cs` | `Star` 프로퍼티로 유닛 성급 조회 가능 |
