## 5단계: Sub-task 도출 ✅

### 5-1. 동작 트리 상세화

```
웨이브 프리뷰
├─ 프리뷰 진입/이탈
│   ├─ 💻 정비 페이즈에서 전환 버튼을 눌러 침략자 프리뷰 카메라로 전환한다
│   ├─ 💻 프리뷰 화면에서 전환 버튼을 눌러 아군 배치 카메라로 복귀한다
│   └─ 💻 전투 시작 시 프리뷰 상태이면 자동으로 아군 카메라로 복귀한다
├─ 프리뷰 화면 구성
│   └─ 💻 실제 침략자 엔티티를 활용하여 프리뷰 화면에 배치·표시한다
├─ 침략자 목록 표시
│   ├─ 💻 다음 웨이브의 침략자를 종류+성급 기준으로 그룹핑하여 목록에 표시한다
│   ├─ 💻 각 항목에 침략자 스프라이트·성급·등장 숫자를 표시한다
│   └─ ❓ 침략자 등장 위치 기준(우상단→하단→왼쪽)으로 정렬 — 정렬 기준 상세 미정
└─ 침략자 상세 정보 조회
    └─ ❓ 목록의 침략자를 클릭하면 스탯 상세 정보를 확인 — 표시 항목 미정
```

### 5-2. 보조 관점 체크

**데이터/속성:**
- 🧩 웨이브별 침략자 구성 데이터 (라운드 에픽 RoundInfoData에서 관리)
- 💻 침략자 그룹핑 로직 (종류+성급 기준 집계)
- ❓ 등장 위치 기준 정렬 상세 (레인 기준인지 등)

**UI/정보 표시:**
- 💻 침략자 프리뷰 목록 UI
- 💻 카메라 전환 버튼 UI
- 💻 침략자 상세 정보 팝업 UI
- ❓ 프리뷰 유도 UI 구체적 형태 미정
- ❓ 스탯 상세 표시 항목 미정

**기반 작업:**
- 💻 프리뷰용 카메라 시스템 (아군 ↔ 침략자 전환)
