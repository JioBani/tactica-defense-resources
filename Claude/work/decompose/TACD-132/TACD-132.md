# TACD-132: [소환터미널 - 스캔]

> 상태: 완료

---

## 5단계: Sub-task 도출 ✅

### 5-1. 동작 트리 상세화

```
스캔 (리롤)
├─ 스캔 실행
│   ├─ 💻 소환수 풀에서 4개를 랜덤으로 선택한다
│   ├─ 💻 선택된 소환수를 터미널 소환 슬롯에 표시한다
│   └─ ❓ 배치 상한에 따라 성급(2성/3성) 등장 확률을 적용한다
├─ 자동 스캔
│   └─ 💻 매 라운드(정비 페이즈) 시작 시 자동으로 스캔을 실행한다
├─ 수동 재스캔
│   ├─ 💻 재스캔 버튼을 누르면 마나를 소비하고 스캔을 실행한다
│   └─ 💻 마나가 부족하면 재스캔이 불가능하다
├─ 스캔 잠금
│   ├─ 💻 스캔 잠금 버튼으로 잠금/해제를 토글한다
│   ├─ 💻 잠금 상태에서 라운드 시작 시 자동 스캔을 건너뛴다
│   └─ 💻 자동 스캔을 건너뛴 후 잠금이 자동 해제된다
└─ 소환수 풀 결정
    └─ 💻 출전 편성 소환술사가 소환할 수 있는 소환수로 풀을 구성한다
```

### 5-2. 보조 관점 체크

**데이터/속성:**
- ❓ 재스캔 비용: 기획서에 "n원"으로만 명시, 구체적 수치 없음
- ❓ 배치 상한별 성급 등장 확률 테이블: 구체적 수치 없음
- 🧩 소환술사별 소환 가능 소환수 목록: 출전 편성 시스템 소관

**UI/정보 표시:**
- 💻 터미널 소환 슬롯에 스캔된 소환수 4개 표시
- 💻 재스캔 버튼 및 비용 표시
- 💻 스캔 잠금 버튼 및 잠금 상태 표시
- 💻 마나 부족 시 재스캔 불가 시각적 피드백

**기반 작업:**
- ✅ 마나 시스템 (TACD-140, 완료)
- ✅ 배치 상한 시스템 (TACD-134, 완료)
