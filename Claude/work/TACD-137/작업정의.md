# TACD-137: [소환터미널 - 스캔] 스캔 잠금 구현

## 작업 내용

소환 터미널에 **스캔 잠금** 기능을 구현한다.

- 스캔 잠금: 라운드 시작 시에 터미널 자동 재스캔을 막는 토글 기능
- 스캔 잠금 상태에서 라운드가 시작되면 재스캔이 진행되지 않고, 스캔 잠금이 자동으로 해제된다

## 완료 정의

1. 소환 터미널 UI에 스캔 잠금/해제 토글 버튼이 존재한다
2. 스캔 잠금 상태에서 정비 페이즈(Maintenance) 진입 시 자동 재스캔이 스킵된다
3. 스캔 스킵 후 잠금이 자동으로 해제된다
4. 수동 재스캔(Reroll 버튼)은 잠금 여부와 무관하게 동작한다
5. 잠금 상태가 UI에 시각적으로 표시된다
