# TACD-209: 승급 시 스탯 강화 적용

## 작업 내용

성급/강화 단계에 따라 소환수의 기본 스탯이 강화되도록 적용한다.

## 현재 상태

코드 인프라는 이미 구현되어 있다:
- `UnitStatSheet.Init()`: `effectiveStar = star + reinforcement`로 `UnitStatsByLevelData.GetStat(kind, effectiveStar)` 호출
- `UpgradeStar()` / `Reinforce()`: `Init()` 재호출하여 스탯 재계산
- `StarStatRecord`: 성급별 값 리스트 + `additionalPerExtraStar` 지원

**문제**: 데이터 에셋(ScriptableObject)에 성급별 차등 값이 입력되지 않아 실질적으로 스탯이 변하지 않음.

## 완료 정의

- 각 소환수의 UnitStatsByLevelData 에셋에 성급별 차등 스탯이 입력된다
- 합성(승급) 후 소환수 스탯이 성급에 맞게 실제로 변경된다
- 강화 단계에 따른 스탯 증가가 적용된다

## 관련 코드

- **UnitStatsByLevelData** (`Assets/Common/Data/Units/UnitStatsByLevel/UnitStatsByLevelData.cs`)
  - `StarStatRecord` 구조체: 성급별 값 리스트 + 초과 증가치
- **UnitStatSheet** (`Assets/Scenes/Battle/Feature/Unit/Unit/UnitStats/UnitStatSheet.cs`)
  - `Init()`, `UpgradeStar()`, `Reinforce()` — 이미 구현 완료
- **에셋 파일**
  - `Assets/Common/Data/Units/UnitStatsByLevel/공뇽 스텟.asset`
  - `Assets/Common/Data/Units/UnitStatsByLevel/수호자 스탯.asset`

## 관련 기획서

- 07 게임 시스템 기획서 > 2.2.4 승급 효과

## 작업 단위

- [ ] 소환수 에셋에 성급별 차등 스탯 데이터 입력
