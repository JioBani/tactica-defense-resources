# 의사결정 기록 (Design Decision Log)

> 이 문서는 **"왜"** 를 기록한다.
> 기획서(00/04/07/08)가 **"무엇(현재 사실·스펙)"** 을 정의한다면, 여기서는 그 결정에 이른 **배경·이유·대안·폐기 사유**를 남긴다.

## 운영 원칙

- 기획서 본문에는 **사실만**, 결정의 *이유*는 이 문서에 분리한다.
- 각 결정 기록은 한 번 적으면 **고치지 않는다(immutable).** 결정이 바뀌면 옛 항목 상태를 `대체됨(→ D-NNN)` 으로 바꾸고 **새 번호로 새 항목**을 추가한다 → 번복 히스토리가 그대로 남는다.
- 번호 `D-NNN` 은 추가 순서대로 부여하며 **재사용·재정렬하지 않는다.** (아래 그룹은 읽기 편의를 위한 분류일 뿐, 번호 순서가 곧 시간 순서)
- 상태: `확정` / `잠정` / `검토중` / `대체됨` / `폐기`.

### 항목 템플릿

```
### D-NNN. 제목
- 상태 / 일자
- 결정: (한 줄)
- 배경: (왜 이 문제가 생겼나)
- 이유: (근거)
- 대안/폐기: (선택)
- 반영 위치: (기획서 어느 챕터에 사실로 들어갔는지)
```

## 제작 철학 (북극성)

### D-001. 게임 제작 목적 = TFT의 3가지 아쉬움 해소 (M-1/M-2/M-3)
- 상태: 확정 / 2026-05
- 결정: 모든 시스템 결정은 다음 3개 동기 부합 여부로 판단한다.
  - **M-1** 빌딩한 팀을 *오래* 즐기고 싶다 (TFT의 PvP 타임어택 아쉬움 → PvE 싱글).
  - **M-2** 가챠·서브컬쳐 매력으로 캐릭터 애정·중독성.
  - **M-3** 반복 플레이의 지루함 극복 (싱글이라 시즌 갱신이 없어 더 취약).
- 이유: 이 게임은 "TFT에서 느낀 내 아쉬움"을 풀려고 만든다. 판단 기준이 흔들리지 않도록 명문화.
- 반영 위치: [[00 기획서]] 1.2 핵심 세일즈 포인트.

### D-002. 수집 대상 = "기물"이 아니라 "시너지(=소환술사)"
- 상태: 확정 / 2026-05
- 결정: 가챠 수집 단위를 소환수(기물)가 아니라 **소환술사(=심상 효과)** 로 둔다.
- 배경: 기물(20~50종)을 모두 매력적 캐릭터로 디자인하면 제작량·유저 인지 부담이 폭증.
- 이유: "시너지 ↔ 학교/심상/소환술사" 로 묶어 **소환술사 = 시너지의 캐릭터화**. 수집 대상 수를 줄이면서 "매력 + 인게임 영향"을 동시 충족. 수집 상태에 따라 빌딩 풀이 달라져 M-3에도 기여.
- 반영 위치: [[00 기획서]] 1.5 용어, 2.1.3.

### D-003. 빌딩 축은 줄이고, 깊이를 늘린다
- 상태: 확정 / 2026-05
- 결정: 유닛 레벨(코스트별 등장확률)·레어도·아이템을 **제거**, 시너지·전술 지원에 결정 깊이를 집중. 배치 상한 증가가 레벨을 대체.
- 이유: 싱글이라 한 선택을 오래 고민할 수 있으니 선택지는 많아도 되지만, **축이 너무 많으면 피로**. 축을 줄이고 각 축을 깊게.
- 반영 위치: [[00 기획서]] 2.1.4.2, [[07 게임 시스템 기획서]] 6·7.

## 전투 모델

### D-010. 전투 = 레인 이동 + 2D 원형 사거리 + 충돌 없음
- 상태: 확정 / 2026-05-30
- 결정: 적은 레인을 따라 R→L 직진(이동은 레인 고정). 사거리는 **레인 무관 2D 원형 반경**. 유닛끼리 **충돌·점유 없음**(겹침 허용). 사거리 내 타겟 생기면 멈춰 공격, 사망 시 재진군. 타겟 우선순위 = 최근접.
- 배경: PvsZ식 "같은 레인 차단"만으로는 역할군의 공간적 의미가 빈약.
- 이유: 사거리가 레인을 가로지르므로 **모든 레인을 1:1로 막을 필요가 없고**, 후방·인접 레인 배치로 커버하는 전략 깊이가 생긴다. 8열 깊이로 전후 배치(앞 탱커/뒤 원거리)가 자연 발생.
- 반영 위치: [[07 게임 시스템 기획서]] 1.6 / 1.7 / 2.3.1.

### D-011. 탱킹 = 위치 기반 어그로 (물리 차단 아님)
- 상태: 확정 / 2026-05-30
- 결정: 충돌이 없으므로 탱커는 길을 막지 않는다. **"가장 가까운 대상"이라 적의 공격을 끌어주는** 식으로 몸빵.
- 이유: 충돌 없음 결정(D-010)의 귀결. 탱커를 앞 열에 세워야 어그로가 작동. (도발/강제 어그로는 추후 스킬로 보강 가능.)
- 반영 위치: [[07 게임 시스템 기획서]] 2.3.1.

### D-012. 소환술사도 소환수와 동일 스탯 체계, 기본 사거리 1
- 상태: 확정 / 2026-05-30
- 결정: 소환술사는 소환수와 같은 능력치·데미지 공식을 쓴다. 성급은 없고 육성(레벨업·심상 각성)으로 강화. 기본 사거리 1(근접 반격). 단 심상 효과로 직접 공격·사거리 부여 등 변형 가능.
- 이유: 술사도 전장에서 HP로 싸우는데 스탯 정의가 없었음. 별도 체계를 만들지 않고 기존 유닛 체계 재사용으로 단순화.
- 반영 위치: [[07 게임 시스템 기획서]] 1.1.1, [[08-2 육성 스케일링]] 4.2.

## 시너지 / 빌딩

### D-020. 역할군 4종 (탱커 / 평타 딜러 / 스킬 딜러 / 서포터)
- 상태: 확정 / 2026-05-30 — (이전 5종[탱커/근딜/원딜/마법사/힐러]을 대체)
- 결정: 역할군을 4종으로. 근딜 폐기, 원딜+마법사를 "딜러"로 통합하되 **평타형/스킬형**으로 구분(스킬형이 강한 스킬로 변수 창출).
- 배경: 근딜 제거 후 원딜·마법사가 둘 다 원거리라 사거리 축 명칭이 겹침 → 구분 축을 "딜 방식(평타/스킬)"으로 이동.
- 반영 위치: [[07 게임 시스템 기획서]] 1.4.

### D-021. 술사당 소환수 6체 + 심상 효과 임계치 2/4/6
- 상태: 확정 / 2026-05-30 — (이전 5체·임계치 2/4/6/8 대체)
- 결정: 술사당 소환수 6체(풀 24), 심상 임계치 2/4/6(3단계). 소환수 특성 임계치 2/3/4 유지.
- 이유: 심상 최대 임계치 6 = 한 술사의 6체를 모두 배치하면 자기 심상 메인 효과가 풀 발동 → **6체와 임계치가 맞물림**.
- 반영 위치: [[07 게임 시스템 기획서]] 5.1.2, [[08-2 육성 스케일링]] 4.3.

### D-022. 분포 유형 기반 (일반형 / 서포터형), 탱커 기본 2체
- 상태: 확정 / 2026-05-30
- 결정: 균일 분포가 아니라 술사마다 분포 유형을 가짐. 모든 술사 탱커 2 고정, 일반형은 딜러로 나머지(평타/스킬 세부), 서포터형은 서포터 1 포함. 단일 딜러 역할군 최대 3.
- 이유: 출전 4명 합산 시 역할군 균형을 맞추는 것이 **빌딩의 첫 결정 단계**가 되도록. 서포터는 희소하게 둬 확보 자체를 전략화.
- 반영 위치: [[07 게임 시스템 기획서]] 2.0.2, [[08-1 컨텐츠 레벨디자인]] 7.9.

### D-023. 심상 효과 = 메인 효과 + 글로벌 버프 이중 구조 ("버림패 술사" 방지)
- 상태: 확정 / 2026-05
- 결정: 심상 효과는 *같은 심상 강화(메인)* + *전체 약한 강화(글로벌 버프)* 의 이중 컴포넌트. 글로벌 버프는 2 임계치에서 발동.
- 배경: 한 게임에 주력 심상 2개로 수렴하면 나머지 2 술사가 "버림패"가 될 위험.
- 이유: 비주력 술사도 소환수 2체만 splash하면 글로벌 버프로 기여 → 4명 모두 가치. **이 2체 투자는 버그가 아니라 의도된 전략적 비용**.
- 반영 위치: [[07 게임 시스템 기획서]] 5.2.1.

### D-024. 소환수 특성 부여 = 종류 단위·전장 내 고정
- 상태: 확정 / 2026-05-30
- 결정: 전장 시작 시 5개 특성을 **소환수 종류**(24종)에 배정, 한 전장 내 고정. 같은 종류는 몇 마리든 같은 특성 → 합성·추가 소환에도 유지. 전장이 바뀌면 재배정.
- 이유: 개체 단위 부여면 합성(같은 종 3체→1체) 시 특성 충돌·승계 문제가 생김. 종류 단위로 못박아 해소. (랜덤 부여 자체는 M-3 다양성 + 학습 부담 감소가 목적.)
- 반영 위치: [[07 게임 시스템 기획서]] 5.3.

## 콘텐츠 / 레벨 디자인

### D-030. V-1 ↔ V-2 충돌 해법 = 정규 라운드 + 엑스트라 라운드
- 상태: 확정 / 2026-05
- 결정: 한 전장을 *정규 라운드(시간 제한 → V-2)* + *엑스트라 라운드(카타르시스 → V-1)* 2단으로.
- 배경: V-1(빌드를 오래 즐김)과 V-2(한 판이 너무 길면 피로)는 라운드 수로 보면 충돌.
- 이유: 정규 라운드는 약 20분으로 마감(V-2), 엑스트라 라운드는 강한 빌드일수록 더 오래 끌며 빌드 진가를 즐김(V-1). "강제로 길게"가 아니라 "원할 때 길게".
- 반영 위치: [[00 기획서]] 6.2, [[08-1 컨텐츠 레벨디자인]] 7.2.

### D-031. 엑스트라 라운드 = 자동 연장 + 언제든 퇴장 (진입 선택 아님)
- 상태: 확정 / 2026-05-30 — (이전 "본편 클리어 후 진입 선택" 모델 대체)
- 결정: 정규 라운드 클리어 시 승리 확정 + 엑스트라 라운드 **자동 시작**. 플레이어는 언제든 "클리어 확정 후 퇴장". 엑스트라 중 패배 = 실패 아님(보상 정산 후 종료). 라운드당 보상 배수 +0.1, 보통 5·최대 10라운드까지 가능하게 급격한 난이도.
- 이유: "진입 여부 선택"보다 "자동 연장 후 빠져나갈 타이밍 선택"이 매끄럽고, 빌드 강도에 비례해 길이가 자연 결정됨.
- 반영 위치: [[07 게임 시스템 기획서]] 4.1.2 / 4.1.3, [[08-1 컨텐츠 레벨디자인]] 7.2~7.4.

### D-032. 방어전 모드 폐기
- 상태: 폐기 / 2026-05-30
- 결정: 별도 "방어전(처음부터 무한 진행)" 모드를 두지 않는다.
- 이유: 엑스트라 라운드가 무한 경험을 캠페인 안에서 제공하므로 별도 모드의 차별화 근거가 약함. 콘텐츠를 캠페인에 집중.
- 반영 위치: (해당 서술을 기획서에서 제거)

## 메타 / 수집

### D-040. 단일 가챠 + "가챠 재화 / 강화 재화" 2종 체계
- 상태: 확정 / 2026-05
- 결정: 소환술사·전술 지원·재화를 **하나의 가챠**로 통합(가챠 재화). 강화는 별도 강화 재화. 술사 각성은 심상조각.
- 이유: 재화 종류 폭증을 막으면서 "뽑기 vs 강화"의 결정 영역을 명확히 분리. (원칙: 뽑기는 같은 재화끼리, 강화는 같은 재화끼리.)
- 반영 위치: [[00 기획서]] 2.1.1, [[08-2 육성 스케일링]] 2·3.

### D-041. 전술 지원 = 출력 핵심 + 유저 친화 알고리즘 보정
- 상태: 확정 / 2026-05
- 결정: 전술 지원이 시너지보다 **출력 우위**. 등장 알고리즘이 현재 시너지·이전 선택에 맞춰 보정되어 빌드 방향에 어울리는 후보가 잘 나오게.
- 이유: 아이템이 없으므로 전술 지원이 TFT의 증강+아이템 역할을 흡수. 시너지(가챠 캐릭터)가 *방향*을, 전술 지원이 *출력*을 담당하되, 알고리즘 보정으로 둘을 자연스럽게 묶어 M-2(캐릭터 매력)를 보호.
- 반영 위치: [[07 게임 시스템 기획서]] 6.

### D-042. 강화 매몰비용 문제는 가챠 매력으로 자연 해소
- 상태: 확정 / 2026-05
- 결정: 별도 매몰비용 완화 장치(자원 호환·신규 술사 보너스)를 우선 두지 않는다.
- 이유: 새 술사를 뽑으면 그 심상 효과가 궁금해 자연스럽게 쓰고 싶어짐 → M-2 자체가 해법. 단 이게 작동하려면 신규 술사 페이스 + 심상 효과 매력이 충분해야 함(콘텐츠 단계 책임).
- 반영 위치: (의도 — 기획서엔 직접 서술 없음)
