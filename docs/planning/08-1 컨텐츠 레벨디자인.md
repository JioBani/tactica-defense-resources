# 컨텐츠 및 레벨디자인

## 7.1 캠페인(전역/전장/보상/레벨디자인)

### 7.1.1 전역 및 전장

- **전역**: 캠페인 모드의 큰 줄기. 전역을 선택하여 포함된 전장을 순차적으로 플레이할 수 있다.
  - 예: 블라디보스토크 전역, 한반도 전역 등
- **전장**: 실제로 플레이하게 되는 인게임 한 판 단위.
- 전역 내의 모든 전장을 클리어 하면 다음 전역을 플레이할 수 있게 된다.
- 전역 및 전장의 플레이에 맞춰 스토리가 진행된다 (캠페인 및 스토리 기획 참고).

### 7.1.2 전역 내부 구성

전역 안에는 다음 종류의 전장이 포함된다.

- **일반 전장**: 표준 난이도·플레이 타임의 전장. 다수.
- **중간 보스 전장**: 일반 전장보다 난이도와 플레이 타임이 더 높을 수 있다. 전역의 중간 지점에 위치.
- **보스 전장**: 전역의 마지막 전장. 가장 높은 난이도·플레이 타임을 가질 수 있다.

세부 전장 수·중간 보스 위치는 콘텐츠 디자인 단계에서 정의한다.

## 7.2 한 전장의 시간 / 라운드 구조

### 7.2.1 디자인 핵심 가치

본 게임의 레벨 디자인은 다음 두 가치를 동시에 만족시키도록 설계한다.

- **V-1. 팀 빌딩 카타르시스 충족**: 빌드가 완성되어 작동하는 순간이 충분히 길게 이어져야 한다.
- **V-2. 한 게임 길이의 적정성**: 한 전장이 과도하게 길면 피로가 누적된다. 정비 페이즈에 결정이 많은 "머리 쓰는 게임" 이므로, 다른 액션 게임보다 길이 제약이 더 엄격하게 작동한다.

두 가치는 본질적으로 충돌한다. 현재는 **본편만**으로 한 전장을 구성하여 V-2(게임 길이 적정성)를 우선 충족한다.

> **(무한 모드 — 추후 도입 보류)** 두 가치의 충돌은 본래 한 전장을 **본편 + 자율 무한 모드** 의 2단 구조로 풀어 동시에 만족시킬 계획이었다. 무한 모드를 일단 제외하면서 **V-1(긴 카타르시스) 충족 수단은 미해결 상태**로 남으며, 무한 모드 재도입 또는 대체 수단이 필요하다. 아래 7.2.2 / 7.2.3 / 7.3 / 7.4 는 재도입 시 복원할 설계 메모이다.

### 7.2.2 본편 + 자율 무한 모드 구조 *(추후 도입 보류)*

| 구간 | 목표 시간 (잠정) | 역할 |
|---|---|---|
| **본편** | 약 20분 | V-2 충족 — 정해진 라운드 수로 한 판이 마감되는 명확한 컷 |
| **무한 모드** | 평균 약 10분 | V-1 자율 충족 — 카타르시스가 부족하면 플레이어가 선택 진입 |

- 본편 라운드 수는 위 시간 목표에 맞춰 결정한다 (한 라운드 ≈ 2분 가정 시 약 10~14 라운드 수준, 실제 플레이 테스트로 보정 필요).
- **강제가 아니라 자율** — 본편만 하고 빠지면 한 게임이 약 20분에 끝나고, 무한 모드까지 가면 약 30분.

> ※ 무한 모드 보류 중에도 **본편 라운드 수 / 한 라운드 ≈ 2분 / 본편 약 20분** 기준은 그대로 유효하다.

### 7.2.3 자율 무한 모드의 효과 *(추후 도입 보류)*

- TFT 에서 카타르시스가 부족했던 이유는 "팀이 완성됐을 때 적이 약해서 금방 끝남" 이다.
- 본 게임은 본편을 압축적인 길이로 두는 대신, 본편 클리어 후 **난이도가 가파르게 상승하는 무한 모드** 를 두어 강한 빌드를 가진 플레이어가 자기 빌드의 진가를 끝까지 즐길 수 있도록 한다.

## 7.3 무한 모드 디자인 규칙 *(추후 도입 보류)*

> 무한 모드는 일단 제외한다. 아래는 재도입 시 복원할 설계 메모이다.

무한 모드는 다음 세 가지 제약을 동시에 만족하도록 설계한다.

### 제약 1. 너무 쉬우면 의미 없음 (가파른 난이도)

- 본편 빌드가 무한 모드 초반을 손쉽게 휩쓸면 V-2 가 무너진다.
- 무한 모드는 **본편 후반보다 명백히 어려운 난이도부터 시작** 하며, 라운드를 거듭할수록 가파르게 상승한다.
- 이는 보상 정책의 자연 cap 으로도 작동한다 (7.4 참고).

### 제약 2. 진입 유인이 있어야 함

플레이어가 본편 클리어 후 무한 모드를 선택할 이유:

- **카타르시스 자체** (가장 본질적): 내가 빌딩한 팀이 어디까지 버티는지 확인.
- **명예 보상**: 최고 도달 라운드 기록 등.
- **보상 배수**: 본편 보상 × 무한 모드 배수 (7.4 참고).

### 제약 3. 부조리하면 안 됨 (예측 가능한 곡선)

- 갑자기 침략자가 100배 강해져서 어떤 빌드든 즉사하는 식의 비약은 금지.
- 난이도 상승 곡선은 **부드럽게 가파른** 형태 (단계적 폭발이 아닌 매끄러운 가파른 상승).
- **빌드를 강화한 만큼 더 오래 버틸 수 있는 비례 관계** — 플레이어 결정의 영향이 명확해야 한다.

## 7.4 보상 정책

- 한 전장의 최종 보상 = **본편 클리어 보상**.

> **(무한 모드 — 추후 도입 보류)** 무한 모드 도입 시 아래 배수·패배 처리·안전장치 규칙이 복원된다.

### 7.4.1 보상 배수 *(추후 도입 보류)*

- 한 전장의 최종 보상 = **본편 클리어 보상 × 무한 모드 배수**.
- 무한 모드 1라운드 클리어마다 **+0.1 배수**.
- 예시:
  - 본편만 클리어 = ×1.0
  - 무한 5라운드 = ×1.5
  - 무한 10라운드 = ×2.0

### 7.4.2 무한 모드 패배 시 처리 *(추후 도입 보류)*

- 무한 모드 진입 시 잔여 패배 횟수는 0으로 리셋된다. 1회 패배 시 즉시 전장 종료.
- 단 이 경우 "임무 실패" 가 아니라, 본편은 이미 클리어한 상태이므로 **그 시점까지의 보상 배수를 그대로 받는 정산** 처리로 종료된다.

### 7.4.3 강제 무한 분위기 방지 *(추후 도입 보류)*

- 무한 모드 보상이 너무 강해 "본편만 하면 손해" 인 분위기가 형성되면 V-2 가 무너진다.
- 안전장치는 두 가지:
  1. **무한 모드의 가파른 난이도** — ×1.5 배수 (5라운드) 도달조차 어렵게 설계. 평균 도달은 1~3 라운드 (≈ ×1.1~1.3 배수) 예상.
  2. **패배 0 리셋** — 빌드가 약하면 무한 모드가 빠르게 종료되어, 강제로 길게 진행하는 것이 구조적으로 불가능.

## 7.5 침략자 레벨 디자인

(추후 정의 — 콘텐츠 디자인 단계)

- 침략자 종류, 능력치 곡선, 진군 패턴, 보스 메커닉 등.

## 7.6 보상 스케일링

(추후 정의 — 콘텐츠 디자인 단계)

- 전장별 클리어 보상 곡선, 캠페인 전체 진행 보상 곡선, 가챠 재화·강화 재화의 흐름 등.

## 7.7 방어전 모드 *(추후 도입 보류)*

- 처음부터 무한 모드로 진행되는 별도 모드.
- 무한 모드 기반이므로 일단 제외한다. 캠페인의 무한 모드와의 차별점·별도 보상 등 자세한 룰은 무한 모드 도입 시 콘텐츠 디자인 단계에서 정의한다.

## 7.8 UX — 일시정지 / 이어하기

- **일시정지**: 한 전장 진행 중 일시정지 가능 (적어도 정비 페이즈).
- **이어하기**: 게임 종료 후 진행 중이던 전장을 이어 진행할 수 있는 기능을 둘 가능성 (미확정).
- 한 전장이 약 20분(무한 모드 도입 시 약 30분)에 이를 수 있으므로 모바일 UX 친화성을 위해 위 기능이 필요하다.
