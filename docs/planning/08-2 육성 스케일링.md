# 육성 스케일링

본 문서는 메타(전장 외부) 강화 시스템의 구조와 재화 흐름을 정의한다.

## 1. 4가지 강화 축

본 게임의 메타 진행은 다음 4가지 축으로 구성된다.

| 강화 축 | 단위 | 시점 | 적용 |
|---|---|---|---|
| **가챠 (소환술사 모집)** | 캐릭터(=심상 효과) 풀 확장 | 메타 | 영구 |
| **소환술사 육성** | 캐릭터의 심상 효과 강화 | 메타 | 영구 |
| **전술 지원 (모집/연구소)** | 인게임 등장 풀 + 효과 강도 | 메타 | 영구 (인게임 등장 시 자동) |
| **출격 모듈** | 인게임 시스템 자체 강화 | 메타 | 영구 (전장마다 자동 적용) |

각 축은 서로 다른 차원에서 작동하므로 역할 중첩이 없다.

## 2. 재화 체계

본 게임은 재화 종류 폭증을 막기 위해 다음 2종 체계를 원칙으로 한다.

| 재화 카테고리 | 용도 |
|---|---|
| **가챠 재화** | 단일 가챠 (소환술사 · 전술 지원 · 재화 통합 풀, 수집형) |
| **강화 재화** | 출격 모듈 강화 · (그 외 강화형 시스템) |

원칙: **"뽑기" 는 같은 재화끼리, "강화" 는 같은 재화끼리.**

### 2.1 소환술사 육성 재화

소환술사 육성은 두 행위로 나뉘며, 재화가 분리된다.

- **심상 각성** (레벨 상한 돌파): **심상조각** 을 사용한다 (중복 가챠로 획득, 캐릭터 귀속). 잉여분은 **공용 심상조각** 으로 전환. → **확정**.
- **레벨업 (강화)**: 수치 강화에 사용하는 재화는 **미결** (강화 재화 통합 vs 별도 재화). 콘텐츠 디자인 단계에서 확정.

## 3. 가챠 시스템

### 3.1 단일 가챠

- 메인 메뉴 → **[모집]** 진입.
- **하나의 가챠 재화** 사용. 1회 / 10회 모집 선택.
- 소환술사 가챠 / 전술 지원 가챠를 분리하지 않고 **하나의 가챠로 통합** 한다. 단일 풀에서 **여러 재화 · 전술 지원 · 소환술사** 등이 함께 등장하며, **소환술사는 낮은 확률** 로 등장한다.
- 소환술사에는 **레어도(희귀 등급) 개념이 없다.** 모든 소환술사의 기본 심상 효과는 동일선상이며, 강도 차이는 육성(레벨업 · 심상 각성)으로만 만든다.

#### 신규 소환술사 획득
- 소환술사 캐릭터 해금. 해당 술사의 심상 효과 사용 가능.
- 덱 빌딩(출전 편성) 화면에서 선택 가능 목록에 추가.

#### 중복 소환술사 획득
- 중복 시 해당 소환술사의 **심상조각** 을 획득한다 → 누적해 **심상 각성**(4.3 참고) 에 사용.
- 이미 만각성했거나 심상조각이 다음 각성분(잠정 3개)을 채운 경우, 잉여 심상조각은 **공용 심상조각** 으로 전환된다.
- **공용 심상조각** 은 일정 개수(잠정 3개)를 모아 **임의의 소환술사 1회 각성** 에 사용할 수 있다.
- ※ 위 개수(3개 등)는 잠정 수치이며 밸런스 단계에서 변동될 수 있다.

#### 전술 지원 / 재화 획득
- 전술 지원은 단일 가챠 결과로 등장하여, 신규 전술 지원을 해금하거나 보유 전술 지원의 강화 자원을 준다.
- 그 외 여러 재화가 가챠 결과에 포함된다.

### 3.2 가챠 풀 구조 (일부 미결)

- 단일 가챠로 통합하는 방향은 **확정.** 가챠 결과 풀에 **정확히 무엇을 포함할지**(재화 종류 · 전술 지원 · 소환술사 비중)는 추후 검토.
- 소환술사는 낮은 확률로 등장하며, "1명씩" 단위로 나올 가능성이 높다.
- 가챠 확률, 천장 시스템, 픽업 등 세부 사항은 콘텐츠 디자인 단계에서 확정.

## 4. 소환술사 육성

소환술사 육성은 **레벨업(강화, 양적)** 과 **심상 각성(질적)** 의 두 축으로 구성된다.

### 4.1 진입 흐름

1. 메인 메뉴 → **[소환술사] / [도감]** 진입
2. 소환술사 목록에서 1명 선택 → 상세 화면 진입

### 4.2 레벨 업 (강화)

- 레벨 업 시 심상 효과의 **수치** 가 상승한다.
  - 예) 같은 심상 소환수의 공격력 +20% → +22%
- 최대 레벨은 **60** 이며, **20레벨마다 레벨 상한** 에 막힌다 (상한 지점: 20 / 40 / 60).
- 상한 해제는 심상 각성(4.3)으로 한다.

### 4.3 심상 각성 (구 '승급' 대체)

- 레벨 상한 도달 후 **심상 각성** 으로 상한을 돌파한다. 총 **3회** (20 / 40 / 60).
- 각성에는 해당 술사의 **심상조각** (또는 **공용 심상조각**)이 필요하다 (잠정 1회당 3개).
- 각성 시 효과:
  - 심상 효과 **자체가 강화** 되어 **추가 효과** 가 부여된다.
    - 예) "공격력 +20%" → 각성 후 "+ 매 공격 시 3초간 공격력 +5%" 추가
- ※ 소환술사에는 **레어도(희귀 등급) 개념이 없다.** 모든 소환술사의 기본 심상 효과는 동일선상이며, 강도 차이는 레벨업·심상 각성으로만 만든다.
- ※ 기존 '승급'에 있던 **시너지 임계치 단계 확장** 효과는 **삭제** 되었다. 심상 효과 임계치는 **2 / 4 / 6 / 8 로 고정** 이다.

## 5. 전술 지원 연구소

### 5.1 진입 흐름

1. 메인 메뉴 → **[전술 지원 연구소]** 진입.
2. 현재 해금한 전술 지원과 적용 상태 확인.

### 5.2 해금 경로

- **가챠 모집** (3.2 참고)
- **캠페인 조건부 해금**:
  - 예: 특정 임무 클리어
  - 예: 특정 심상 계열 소환술사 N명 획득

### 5.3 강화

- 강화 재화를 사용해 보유 전술 지원의 효과 강도를 강화.
- 강화된 전술 지원은 인게임 중 등장 후보 풀에 자동으로 반영된다 (별도 장착 단계 없음).

## 6. 출격 모듈 강화

### 6.1 진입 흐름

1. 메인 메뉴 → **[출격 모듈]** 진입.
2. 해금된 출격 모듈 목록 확인.

### 6.2 해금 경로

- 캠페인 진행 보상 등 조건부 해금.

### 6.3 강화

- **강화 재화** 를 사용해 출격 모듈을 강화.
- 보유한 모든 출격 모듈은 한 전장 시작 시 자동 적용된다 (별도 장착 단계 없음).
- 출격 모듈에는 패시브형 (상시 효과) 과 단발성 액티브형 (특정 조건/타이밍 발동) 이 모두 존재한다.

## 7. 후속 정의 필요 항목

다음은 본 문서의 다음 개정에서 정의해야 할 항목들이다.

- 레벨업(강화) 재화 체계 (강화 재화 통합 vs 별도 재화) — 각성 재화는 심상조각으로 확정
- 단일 가챠 결과 풀 구성 (포함 재화 종류 · 전술 지원 · 소환술사 비중)
- 한 번에 등장하는 소환술사 수 (1명 가챠 vs 여러 명 가챠)
- 가챠 확률 / 천장 시스템 / 픽업
- 심상 각성 추가 효과 카탈로그
- 강화 곡선 (레벨업 비용·심상 각성 비용·출격 모듈 강화 비용)
- 심상조각 / 공용 심상조각 전환·소모 수치 (잠정 3개)
- 캠페인 진행에 따른 재화 획득 곡선
