# 1. 게임 개요

<a id="pitch"></a>
## 1.1 한 줄 소개(피치)

매력적인 캐릭터를 육성 할 수 있는 오토배틀러 디펜스 게임

<a id="core-fun"></a>

## 1.2 핵심 세일즈 포인트

#### 1) 오토배틀러에 수집·육성 컨텐츠 결합

- 가챠를 통해 **매력적인 캐릭터와 함께 ‘게임에 등장 가능한 시너지(특성)’**를 획득
- 획득한 시너지는 **육성**할 수 있으며, 육성한 시너지로 **이번 런에 등장할 시너지 풀/구성을 빌딩** 가능
- 수집·육성의 성과가 곧 **조합 선택지 확장**으로 이어져 장기 플레이 동기 강화

#### 2) 덱 빌딩 기반 전략 다변화로 반복 플레이 지루함 감소

- 기존 오토배틀러는 메타가 굳을수록 **조합이 고착화되어 반복감**이 커지는 단점이 있음
- 본 게임은 보유 시너지에 따라 **플레이어가 이번 런의 등장 시너지 풀/구성을 직접 설계**
- 시너지 빌딩 결과에 따라 매 게임 **조합 경로와 운영 판단**이 달라져 반복 플레이의 지루함을 완화

#### 3) PvE 구조로 ‘완성된 조합’의 재미를 끝까지 경험

- PvP 오토배틀러는 고도화된 조합이 완성되기 전에 **게임이 종료되는 경우**가 많음
- 본 게임은 PvE 기반으로, 진행 중 **다양한 버프/강화 선택지**를 제공해 전략을 단계적으로 고도화
- 완성된 팀 조합과 시너지 운용의 재미를 **충분한 플레이 구간에서 체감**하도록 설계

<a id="references"></a>

## 1.3 레퍼런스 정리(TFT 요소 / PvsZ 요소)(작성중)

<a id="win-lose"></a>

### 1) TFT(Teamfight Tactics)에서 차용하는 요소

#### (A) 그대로 채용한 요소

- **상점(Shop) 기반 덱 빌딩 루프**
   매 라운드 골드를 사용해 유닛을 구매/판매하고, 리롤을 통해 원하는 구성을 맞춰가는 흐름을 채용한다.
- **시너지(특성) 기반 팀 구성**
   특정 조합 조건을 만족하면 전투 보너스를 얻는 “시너지 임계치” 구조를 채용한다.
- **유닛 합성(중복 구매 → 승급)**
   동일 유닛을 모아 상위 단계(2성/3성 등)로 승급시키며 파워 스파이크를 만드는 재미를 채용한다.

#### (B) 본 게임에 맞게 변주한 요소

- **레벨업 시스템의 재정의(확률표 중심 → 해금 중심)**
   TFT처럼 “레벨업에 따른 코스트별 등장 확률 변화”를 핵심 규칙으로 두지 않고,
   **배치 확장(골드 구매)** 을 통해 **배치 상한을 늘리고 상위 티어 기물을 해금**하는 방식으로 변주한다.
- **시너지의 ‘랜덤 등장’이 아니라 ‘사전 덱 빌딩’ 기반 등장**
   덱 빌딩 단계에서 **시너지 4개를 선택**하고, 해당 시너지에 속한 기물만 런에서 등장하도록 제한한다.
   → 런의 전략 방향을 “상점 운”보다 “사전 선택(덱 구성)”에 더 강하게 연결한다.
- **기물 풀의 구조화(시너지당 8종: 3 / 2 / 2 / 1)**
   선택한 시너지 1개당 기물 8종이 등장하며, 티어 분포는 **1티어 3 / 2티어 2 / 3티어 2 / 4티어 1**로 고정한다.
   특히 **4티어는 ‘시그니처 캐릭터’ 1종이 직접 등장**하는 형태로 설계해, 덱의 정점(목표)을 명확히 만든다.

------

### 2) PvsZ(Plants vs Zombies)에서 차용하는 요소

#### (A) 그대로 채용한 요소

- **수비 측 고정 배치**
   아군 유닛은 전투 중 이동하지 않으며, “어디에 배치하느냐”가 핵심 전략이 된다.
- **공격 측 진군(웨이브 기반 진행)**
   적은 웨이브에 따라 아군 진영으로 몰려오며, 시간이 갈수록 압박이 증가한다.
- **레인 기반 디펜스 전투 감각**
   레인/사거리/탱킹/타겟팅 등 배치에 따른 결과가 전투를 좌우하는 “배치 퍼즐” 감각을 채용한다.

#### (B) 본 게임에 맞게 변주한 요소

- **배치 디펜스의 ‘준비 과정’을 오토배틀러식 상점/합성으로 대체**
   PvsZ의 설치형 운영 대신, 상점·리롤·합성으로 전력을 준비하고 웨이브를 넘기는 구조로 변주한다.
- **웨이브 대응의 중심을 ‘덱(시너지) 설계 + 배치 최적화’로 이동**
   스테이지/웨이브 성격에 맞춰 덱 구성(시너지 선택)과 배치를 조정하는 형태로 전략 축을 재구성한다.

------

### 3) 결합 포인트 (본 게임에서의 해석)

- **TFT의 상점·합성·시너지**로 “런마다 조합을 설계”하고,
- **PvsZ의 고정 배치·진군 웨이브**로 “전투를 배치 퍼즐로 해결”한다.
- 즉, **사전 덱 빌딩(시너지 4개 선택) → 상점 운영/합성으로 전력 구성 → 고정 배치로 웨이브 대응**의 3단 구조로 두 레퍼런스를 결합한다.

# 2. 게임 요소 개요

## 2.1. 요소

### 2.1.1 수집 / 육성

- **가챠**: 뽑기 시스템을 통해 전장에 등장할 **시너지(=소환술사)** 를 획득
  - 각 시너지는 해당 시너지를 대표하는 **캐릭터(소환술사)** 와 함께 제공
- **소환술사 육성**: 획득한 소환술사를 성장시켜, **전장**에서 해당 시너지의 성능을 강화
- **전술 지원(인게임 버프) 강화**: 재화를 사용해 **전장**에서 적용되는 효과를 획득·강화
  - 예) 시작 마나 +n, 시작 새로고침(리롤) +n, 초기 골드/상점 옵션 강화 등

### 2.1.2 전선(= 스테이지)

- **캠페인** : 전투의 끝이 존재하며, 전장을 클리어 할 때 마다 스토리가 진행됨
- **방어전** : 무한 모드

### 2.1.3  출전 편성(= 덱 빌딩)

- **편성 방식**: 전장 입장 전, 보유한 **소환술사(시너지) 4명**을 선택해 출전 편성을 구성
- **등장 기물** : 선택한 소환술사가 소환 가능한 소환수가 **소환 터미널**에 등장
  - 각 소환술사마다 8체, 4 * 8 = 32체의 소환수가 등장

### 2.1.4. 전장(=인게임)

#### 2.1.4.1. 전투방식

- **고정 배치 디펜스**: 소환수는 전투 중 이동하지 않음
- **웨이브 진행**: 적(침략자)은 웨이브 단위로 진군하며, 스테이지 진행에 따라 압박이 증가

#### 2.1.4.2. 팀 강화 방식

- **소환수 소환/ 리롤/ 판매**
  - 소환수의 희귀도(코스트) 분리는 존재하지 않음
- **소환수 합성**
- **소환 한계 확장**
- **시너지**
  - **계열** : 소환술사가 가지고 있는 시너지
  - **전술 특성** : 전장마다 랜덤으로 부여되는 시너지
    - 이전 전장에서는 A, B 유닛이 강습 전술 특성을 가지고 있었으나, 다음 전장에서는  C D 유닛이 강습 전술 특성을 가질 수 있음
- **전술 지원** : 전장 진행 중 특정 시점에 버프/강화 선택지를 제공하여 운영 방향과 전투 양상을 변화
- **출격 모듈**(?) : 전장 시작 시 적용되는 보너스를 통해 초반 운영을 보조
  - (예: 시작 마나, 시작 갱신 횟수 등)

#### 전장 진입

1. 로딩 후 전장 입장
2. **출격 모듈** 적용(전장 시작 보너스 자동 반영)
3. 초기 마나/초기 소환 슬롯(배치 한계) 확인
4. 소환수 시너지 분표 표 확인
   1. 어떤 조합을 사용 할 수 있을지 전략 수립

#### 웨이브 진행(전투 루프)

##### 1. 첫번째 정비 페이즈 시작

1. 소환수 시너지 분표 표 확인
   1. 어떤 조합을 사용 할 수 있을지 전략 수립
2. **소환 터미널** 오픈 → 소환수 목록 확인
3. 마나를 사용해 소환수 **소환**
4. 소환수는 대기석에 배치됨
5. 대기석에 배치 된 소환수를 전장에 다시 배치
6. 웨이브 시작 터치

##### 2. 첫번째 전투 페이즈 시작

1. 적(침략자) 진군, 아군(소환수)은 고정 배치로 교전
2. 소환수의 HP가 0이 되면 다운 상태로 전환, 비활성화 되면 침략자는 무시하고 전진함
3. 모든 침략자 웨이브를 막아내면 전투 페이즈 종료

##### 3. 첫번째 전투 페이즈 종료 - 두번째 정비 페이즈 시작

1. 마나 이자 획득
2. 소환수 추가 소환
   1. 동일 소환수 3개 소환하면 자동 합성
3. 필요 시 소환 터미널 **갱신/잠금/배치 상한 추가**
4. 특정 웨이브(또는 체크포인트) 도달 시 **전술 지원** 선택(3택 1 등, 컨셉 확정 필요)

##### 4. 클리어/실패 및 정산

1. 침략자가 한 라인의 끝까지 도착하면, 소환술사와 직접 교전
2. 소환술사도 공격을 하며, 피격에 따라 HP 감소
3. 한명의 소환술사라도 HP가 0이 되면 전장 패배
4. 전장 종료 후 정산 화면 표시
   1. 획득 보상/재화 표시
   2. 소환술사 육성 재료 및 전술 지원/출격 모듈 관련 재화 반영
5. 로비 복귀 → 다음 전장 선택 또는 재도전

## 3.2. 육성 시나리오

### 3.2.1. 소환술사 모집

#### 로비 진입 흐름

2. 배너 선택
3. 모집 재화 확인 후 **1회 / 10회 모집** 선택
4. 결과 연출 → 획득 목록 표시

#### 모집 결과 처리

- **신규 소환술사 획득**
  - 소환술사(캐릭터) 해금 + 해당 소환술사의 **시너지 사용 가능**
  - 덱 빌딩(출전 편성) 화면에서 **선택 가능 목록**에 추가
- **중복 소환술사 획득**
  - 중복 캐릭터는 대신 심상조각 획득
  - 누적 조각으로 소환술사 승급

### 3.2.2. 소환술사 육성

1. 메인 메뉴 → **[소환술사] / [도감]** 진입
2. 소환술사 목록에서 1명 선택 -> 상세화면 진입
3. 레벨 업
   1. 레벨 업을 클릭해서 강화
   2. 레벨업을 하면 시너지 효과가 상승함
      - 예) 기계 특성 소환수 공격력 20 -> 공격력 22
   3. 레벨은 승급에 따라 한계치가 있음
4. 승급
   1. 레벨이 한계치에 도달한 경우 승급 가능
   2. 승급을 위해서는 해당 술사의 심상조각이 필요
   3. 승급을 눌러 심상 조각을 사용해서 승급
   4. 승급한 경우 시너지의 최대치가 상승 및 레벨 상한이 상승
      1. 2 / 3 / 4 -> 2 / 3 / 4 / 5

### 3.2.3. 전술 지원 획득 / 강화

#### 로비 진입 흐름

1. 메인 메뉴 → **[전술 지원 연구소]** 진입
2. 현재 해금한 전술 지원과 적용중인 전술 지원 확인 가능
3. 전술 지원은 특정 조건에 따라 해금
   1. 예) 엘프 계열 소환술사 3명 획득
   2. 임무 3 - 4 클리어 등
4. 인게임 재화를 사용해서 해금된 전술 지원을 연구(구매) 및 강화 가능
5. 획득한 전술 지원은 선택할 필요 없이 모두 자동 적용

아래는 내가 적용해둔 **상세 목차 뼈대(4장~15장 + 부록)**만 깔끔하게 뽑은 거야.

# 4. 전장(인게임) 시스템 상세

![전장 화면](./기획서_업데이트2.assets/전장 화면.png)

## 4.1. 전장 레인/그리드 규칙

#### 전장 구성 요소

- **레인(Lane)**: 적(침략자)이 진군하는 경로. 전장은 4개의 레인으로 구성된다.
- **그리드(Grid)**: 아군 **소환수(수호자)** 를 배치하는 칸 단위 공간.
- **전선(Frontline)**: 레인별로 적이 진입하는 방향(침공 방향)과 아군 방어 방향이 고정된다.
- **소환술사** : 레인의 시작점에는 출격한 소환술사가 한명씩 총 4명 배치된다

#### 배치 영역 규칙(아군 소환수)

- 아군 소환수는 **그리드 칸**에만 배치 가능하다.
- 아군 소환수는 전투 중 **이동하지 않는다(고정 배치)**.
- 한 칸에는 기본적으로 **1소환수**만 배치 가능하다.
- 배치 가능 소환수 수는 **배치 상한**에 의해 제한된다.

------

## 4.2. 라운드 및 승리/패배 조건

### 4.2.1. 라운드의 구성

- 전장은 **정비 페이즈(준비)**와 **전투 페이즈(웨이브)**가 교차한다.
- 정비 페이즈에서 정비를 마치고 전투 시작을 터치하면 전투 페이즈가 시작한다

### 4.2.2. 정비 페이즈 구성

정비 페이즈에서는 아래와 같은 행동이 가능하다.

각 행동에 대한 자세한 내용은 - 4.4 ~ 4.7 참고

#### 1) 소환 터미널(상점) 이용

- **소환수 목록 확인**: 이번 정비 페이즈에 제시된 소환수 라인업을 확인한다.
- **소환(구매)**: 마나(재화)를 소모해 소환수를 획득한다.
- **리롤(새로고침)**: 마나를 소모해 소환수 목록을 갱신한다.
- **잠금(고정)**: 다음 정비 페이즈까지 현재 목록을 유지한다.
- **환원(판매)**: 보유한 소환수를 환원하여 재화(또는 일부 환급)를 얻는다.
- **배치 상한 증가**: 전장에 배치 할 수 있는 소환수의 상한을 증가시킨다.

#### 2) 전장 배치/교체/합성

- **소환수 배치**: 보유한 소환수를 그리드 칸에 배치한다.
- **소환수 이동/스왑**: 소환수의 위치를 자유롭게 변경할 수 있다.
- **대기 슬롯(벤치) 관리**: 전장에 올리지 않은 소환수는 벤치(대기 공간)에 보관된다.
- **소환수 합성**: 합성 규칙을 만족하면 자동으로 합성된다.

#### 3) 전술 지원 선택

- **전술 지원 제시 확인 / 선택** : 특정 웨이브에서 제시된 전술 지원을 확인한다.

#### 4) 다음 웨이브 정보 확인

- **웨이브 프리뷰 확인**: 다음 웨이브의 적 구성을 확인한다.

#### 5) 전투 시작

- **전투 시작**: 정비가 끝나면 전투 시작을 터치해 전투 페이즈로 진입한다.

### 4.2.3. 전투 페이즈 구성

전투 페이즈에서는 **정비 페이즈에서 배치한 소환수** 가 자동으로 전투를 수행하고, **적(침략자)** 은 웨이브 규칙에 따라 레인으로 진군한다.

#### 전투 진행

- **전투 시작**: 정비 페이즈에서 전투 시작을 터치하면 웨이브가 시작된다.
- **적 스폰**: 웨이브 구성에 따라 레인별로 적이 생성되어 진군한다.
  - 세부 내용은 **4.8 전투 페이즈: 침략자 등장(스폰) 규칙** 참고
- **소환수 자동 전투**: 배치된 소환수는 이동 없이 제자리에서 공격/스킬을 사용한다.
- **타겟팅/스킬 발동**: 소환수의 공격 및 스킬은 각 소환수의 규칙에 따라 자동으로 처리된다.
- **시너지 적용**: 활성화된 시너지 효과가 전투 중 계속 적용된다.
- **전술 지원 적용**: 선택한 전술 지원 효과가 전투 중 적용된다.

#### 전투 중 제한 사항

- **배치 변경 불가**: 전투 페이즈 동안 소환수의 위치 변경/교체는 불가능하다.
- **소환 터미널 이용**: 전투 중에 소환, 리롤, 배치 상한 증가는 가능하지만, 전장에 배치는 불가능하다

#### 전투 페이즈 종료 조건

- **전투 승리** : **적 전멸**: 등장하는 모든 적을 처치하면 전투 페이즈가 종료된다.
  - 세부 내용은 **4.8 전투 페이즈: 침략자 등장(스폰) 규칙** 참고
- **전투 패배** : 소환술사중 한명이라도 다운이 되면 전투 패배

#### 전투 페이즈 종료 및 다음 단계

- **다음 정비 페이즈 진입**:  정비 페이즈로 전환된다.

# 16. 세계관 및 배경 설정

<a id="worldview-summary"></a>

## 16.1 세계관 요약

205X년, 외계체의 침략으로 인류가 붕괴 직전까지 몰린 근미래. 인류는 "심상(인지)"을 무기로 바꾸는 기술을 개발했고, 이를 다루는 소환술사와 인류가 운영하는 분산형 거점 도시 네트워크(바스티온)가 최전선이 된다.

<a id="era-genre"></a>

## 16.2 시대 및 장르

- **시대 배경**: 205X년
- **장르**: 학원도시물, 근미래 SF

<a id="story-overview"></a>

## 16.3 스토리 개요

- 근미래, 지구에 외계 생명체(이하 '외계체')가 침략해 인류의 절반이 사망
- 외계체는 북반구의 절반 이상을 점령한 뒤 남하 중
- 인류는 남은 권역에서 치열하게 방어 중

<a id="materialization-origin"></a>

## 16.4 실체화 기술의 기원: 외계체의 하이브

- 외계체는 **하이브(Hive)**라 불리는 공세 거점을 통해 지구를 침략
- 하이브는 내부에 블랙홀을 보유하고 있으며, 이 블랙홀의 질량을 소모해 **페이즈 입자(Phase Particle)**로 변환
- 변환된 페이즈 입자를 이용해 외계체 개체들을 **실체화(Materialization)**시켜 끊임없이 소환하며 공격을 펼침
- 인류는 외계체의 침공 과정에서 이 **실체화 기술**의 존재를 발견하고, 역설계를 통해 자체적인 실체화 시스템을 개발하는 데 성공

<a id="human-response"></a>

## 16.5 인류의 대응: "실체화 무기"와 소환술사

- 인류는 외계체로부터 얻은 실체화 기술을 응용하여, 인간의 인지 능력(심상)을 기반으로 **실체화되는 무기/개체**를 만들어내는 기술을 개발
- 이 실체화를 수행하기 위한 매개(시전자)를 **소환술사(Summoner)**라고 부름
- 소환술사는 자신의 심상을 바탕으로 실체화되는 개체를 생성
  - 실체화된 개체는 술사에 따라 지능을 가질 수도 있고, 술사의 명령을 따름
  - 술사마다 소환 가능한 개체의 형태는 천차만별 (기계, 용, 동물 등 "상상 가능한 것" 전반)

<a id="summoner-qualities"></a>

## 16.6 소환술사의 핵심 자질: 상상력(공상력)

- 술사에게 가장 중요한 것은 **상상/공상력**, 즉 현실과 무관하게 자신의 심상에 깊게 몰입할 수 있는 능력
- 사회화 경험이 많은 성인은 "현실"에 더 강하게 묶여 있어 심상 몰입이 어렵고, 그만큼 뛰어난 술사가 되기 힘듦
- 따라서 소환술사는 주로 **청소년 ~ 20대 초반**의 청년층이 수행

<a id="bastion-network"></a>

## 16.7 바스티온 네트워크(학원도시): 분산형 거점 도시 체계

- 거점 도시는 공식적으로 **바스티온(Bastion)**이라 불림
- 모든 거점 도시는 **바스티온 + 지명**으로 부름
  - 예: 바스티온 서울 / 바스티온 도쿄 / 바스티온 부산
- "학원도시"는 고유명사가 아니라, 바스티온 내부에 소환술사 양성 학교와 학생 거주 구역이 함께 존재하기 때문에 붙은 **도시의 성격(통칭)**
- 각 바스티온은 위치/전황/임무에 따라 방어·출격·연구·보급 비중이 달라지며, 그 특성에 맞춰 운영됨

<a id="school-life"></a>

### 16.7.1 학교 생활: 심상 강화의 핵심 장치

- 술사들은 바스티온에서 거주하며, **비슷한 테마(심상)를 가진 학생들끼리** 모여 학교 생활을 함
  - 예: 기계/공학 심상을 공유하는 학교
  - 예: 판타지 세계관의 정령/마법 심상을 공유하는 학교
- 현실에 존재하지 않는 개념이라도, 상상력이 뛰어난 청소년들이 "유사한 심상"을 공유하는 또래와 함께 생활함으로써 심상을 강화하고 실체화의 안정성과 출력이 높아짐

<a id="energy-source"></a>

## 16.8 에너지원: 페이즈 입자(마나)

- 실체화를 위해서는 숨겨진 쿼크 섹터의 **페이즈 쿼크(=마나)**가 필요
- 본부는 미니 블랙홀을 통해 질량을 소모하고, 이를 페이즈 쿼크(마나)로 변환해 공급

<a id="equipment-plgu"></a>

## 16.9 장비: PLGU(완드)

- 소환술사는 **Phase-Lock Guidance Unit(PLGU/PLU, 위상 고정 유도 장치)**를 사용. 속칭 **완드(Wand)**
- 완드는 두 가지 역할을 가짐
  1. 본부로부터 전송받는 마나(페이즈 쿼크)의 유도
  2. 술사가 실체화하려는 개체의 정보를 기기 내부에 저장/안정화하여, 실체화를 보조

<a id="combat-participation"></a>

## 16.10 전투 참여 방식

- 술사는 완드를 통해 본부로부터 마나를 공급받고, 완드의 보조에 따라 개체를 실체화
- 술사는 실체화한 개체(무기/소환체)를 운용해 전투에 참여

# 17. 용어 사전

- **전장**: 플레이어가 스테이지에 진입해 실제 전투(웨이브)를 진행하는 *한 판(런)* 단위의 인게임 플레이 구간.
- **소환 터미널**(=오토배틀러의 상점): 전장 내에서 소환술사가 소환 가능한 소환수(기물) 목록을 확인하고, **마나를 소모해 소환**하는 기능(기존 ‘상점’)을 의미한다.
  - 소환 터미널에서는 **목록 갱신(새로고침)**, **소환(구매)**, **환원(판매)**, **잠금(고정)** 등의 조작이 가능하다.
