# 1단계 (지금 하는 것)

## 키 비주얼 스타일 시트

목적:

> "우리 게임은 이렇게 생겼습니다"

를 정의하는 것

여기엔

- 대표 캐릭터 1명
- 정면/후면
- SD 버전
- 무기
- 학생증
- UI 샘플
- 컬러 팔레트
- 이펙트 샘플

정도만 들어감.

이걸 보고

- 선 굵기
- 색감
- UI 방향
- SD 비율
- 세계관 느낌

을 결정하는 거야.

------

예시로 말하면

블루아카이브 개발 초기에

"아비도스 학생회장"

캐릭터를 만들었다고 치자.

그걸로

- 교복 방향
- 총기 디자인
- UI
- SD

를 다 검증하는 거지.

아직 다른 캐릭터는 없음.



# 2단계

## 마스터 스타일 가이드

키 비주얼 시트에서 확정된 걸 정리.

```
선 굵기

눈 디자인

교복 규칙

UI 규칙

컬러 규칙

학생증 규칙
```

같은 것들.

------

# 3단계

## 캐릭터 개별 시트

그때부터

주인공

친구

라이벌

선생님

이런 애들을 만듦.

그리고 이때는

```
정면

후면

표정

무기

설정

컬러
```

같은 걸 넣는 거야.

------

그래서 현재 네 작업 순서는

```
[지금]

키 비주얼 스타일 시트
        ↓
스타일 확정
        ↓
마스터 가이드
        ↓
캐릭터 개별 시트
```
