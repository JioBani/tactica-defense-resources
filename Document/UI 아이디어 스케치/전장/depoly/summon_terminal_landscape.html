🎮 이 화면은 무엇인가요?
스마트폰을 가로로 돌렸을 때 보이는 소환 터미널의 UI 레이아웃 데모입니다. 택티카 디펜스는 가로 모드로 플레이하는 디펜스 게임입니다.
📐 레이아웃 구조
상단 60%: 전장 화면 (4개 레인, 시너지, 웨이브 정보)
하단 40%: 소환 터미널 (슬라이드 업/다운 가능)
🎯 소환 터미널 구성
좌측 영역:
• 현재 마나: 💎 125
• 다음 수입: 📈 +15
• 재스캔 버튼 (2G)
• 잠금 버튼
• 배치 확장 버튼 (4G)

중앙 영역:
• 소환 가능한 유닛 슬롯 (가로 스크롤)
• 각 슬롯: 유닛 아바타, 이름, 티어, 비용
• 잠금된 슬롯 표시 (🔒)

우측 영역:
• 전장 배치: 5/8
• 대기열: 3/8
• 재스캔 가능 횟수
💡 인터랙션
• 상단 핸들 바 터치 → 터미널 접기/펼치기
• 유닛 슬롯 탭 → 소환 (마나 차감)
• 좌우 스와이프 → 더 많은 유닛 확인
• 잠금 버튼 → 모든 슬롯 고정
• 재스캔 → 잠금되지 않은 슬롯만 새로고침
🎨 디자인 특징
• 가로 모드 최적화 (16:9 비율)
• 양손 조작 편의성 고려
• 전장과 터미널을 동시에 확인 가능
• 최소화 시 전장 화면 최대화
