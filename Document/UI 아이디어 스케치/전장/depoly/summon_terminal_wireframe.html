📋 UI 구성 요소 설명
⋮⋮⋮ 드래그 핸들
터미널을 열고 닫을 수 있는 인터랙션 포인트
  • 터치/드래그로 터미널 위치 조정
  • 클릭 시 터미널 펼침/접힘 토글
  • 시각적 피드백 제공
💎 자원 표시 영역
현재 마나와 다음 라운드 수입을 한눈에 확인
  • 현재 마나: 소환에 사용 가능한 재화
  • 다음 수입: 다음 라운드에 받을 마나
  • 이자 계산 정보 툴팁 제공 가능
🔄 재스캔 (갱신)
마나를 소모하여 소환 목록을 새로고침
  • 비용: 2골드 (기본)
  • 잠금된 슬롯은 유지
  • 갱신권 보유 시 무료 사용 가능
🔒 스캔 잠금
현재 소환 목록을 다음 라운드까지 고정
  • 무료 기능
  • 개별 슬롯 잠금 또는 전체 잠금
  • 잠금 상태 시각적 표시
📈 배치 확장
전장에 배치할 수 있는 소환수 수 증가
  • 비용: 4골드 (기본)
  • 레벨업 개념과 유사
  • 최대 배치 상한까지 확장 가능
🎴 소환 슬롯
소환 가능한 유닛 목록 (기본 5개)
  • 유닛 초상화 (아이콘/이미지)
  • 유닛 이름
  • 소속 시너지 뱃지
  • 티어 표시
  • 소환 비용 (마나)
  • 클릭 시 소환 (마나 차감)
📊 배치 현황
전장/대기열 상태 및 통계 정보
  • 배치 현황: 전장 배치된 유닛 수
  • 대기열: 벤치의 유닛 수
  • 갱신 가능: 남은 갱신권 또는 무제한
⚡ 빠른 액션
자주 사용하는 기능에 빠르게 접근
  • 전체 환원: 모든 유닛 일괄 판매
  • 시너지 보기: 시너지 상세 패널 열기
  • 추천 조합: AI 추천 기능 (선택)

🔄 주요 인터랙션 플로우
1️⃣ 소환 (구매)
조건: 마나 충분 + 대기열/배치 공간 있음
  • 슬롯 클릭 → 마나 차감
  • 유닛이 대기열(벤치)로 이동
  • 슬롯이 비워지고 새 유닛 표시 (갱신)
  • 배치 현황 UI 업데이트
2️⃣ 재스캔 (갱신)
조건: 마나 2G 이상 또는 갱신권 보유
  • 재스캔 버튼 클릭
  • 마나 2G 차감 (또는 갱신권 1 소모)
  • 잠금되지 않은 슬롯만 새로고침
  • 새 유닛 목록 표시
3️⃣ 잠금 (고정)
조건: 없음 (무료)
  • 잠금 버튼 클릭 (전체 잠금)
  • 또는 슬롯 개별 잠금 (홀드/롱프레스)
  • 잠금 아이콘 표시
  • 다음 라운드까지 목록 유지
4️⃣ 배치 확장
조건: 마나 4G 이상 + 최대 레벨 미만
  • 배치 확장 버튼 클릭
  • 마나 4G 차감
  • 배치 상한 +1
  • 배치 현황 UI 업데이트
  • 상위 티어 유닛 해금 (선택적)
5️⃣ 환원 (판매)
조건: 대기열 또는 전장에 유닛 존재
  • 유닛을 드래그하여 환원 영역으로
  • 또는 유닛 우클릭/롱프레스 → 환원 선택
  • 환원 가치만큼 마나 획득
  • 유닛 제거
6️⃣ 터미널 토글
조건: 언제든지 가능
  • 드래그 핸들 클릭
  • 또는 전용 버튼 클릭
  • 터미널 슬라이드 업/다운 애니메이션
  • 전투 중에도 열람 가능 (구매만 제한 또는 전체 가능)

💡 추가 고려사항
🎮 UX 개선 포인트
  • 드래그 앤 드롭: 유닛을 전장에 바로 배치
  • 더블탭: 빠른 소환 (대기열로)
  • 홀드: 유닛 상세 정보 툴팁
  • 스와이프: 터미널 열기/닫기
  • 진동 피드백: 액션 성공/실패 시
🎯 시각적 피드백
  • 마나 부족 시: 슬롯 흔들림 + 경고 색상
  • 소환 성공: 골드 파티클 효과
  • 시너지 매칭: 슬롯 테두리 하이라이트
  • 추천 유닛: 별 아이콘 또는 빛 효과
  • 새 유닛 등장: 페이드인 애니메이션
⚙️ 개인화 옵션
  • 자동 구매: 특정 유닛 자동 소환
  • 자동 합성: 3개 모이면 자동 승급
  • 슬롯 개수: 5개/7개/10개 선택
  • 정렬 기준: 티어/시너지/이름/코스트
  • 터미널 위치: 하단/측면 선택
📱 모바일 최적화
  • 터치 영역 최소 44x44px
  • 스와이프 제스처 지원
  • 한손 조작 고려한 버튼 배치
  • 가로/세로 모드 대응
  • 노치/펀치홀 영역 회피
♿ 접근성
  • 색맹 모드: 색상 외 아이콘으로 구분
  • 폰트 크기 조정 옵션
  • 고대비 모드
  • TTS (화면 읽기) 지원
  • 단축키 지원 (PC 환경)
🔧 기술적 구현
  • 터미널 상태 관리 (열림/닫힘/최소화)
  • 슬롯 풀링 시스템 (성능)
  • 애니메이션 최적화 (GPU 가속)
  • 네트워크 동기화 (온라인 모드)
  • 로컬 저장: 잠금 상태/설정 유지
