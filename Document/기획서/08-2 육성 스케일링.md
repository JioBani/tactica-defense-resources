# 육성 스케일링

본 문서는 메타(전장 외부) 강화 시스템의 구조와 재화 흐름을 정의한다.

## 1. 4가지 강화 축

본 게임의 메타 진행은 다음 4가지 축으로 구성된다.

| 강화 축 | 단위 | 시점 | 적용 |
|---|---|---|---|
| **가챠 (소환술사 모집)** | 캐릭터(=심상 효과) 풀 확장 | 메타 | 영구 |
| **소환술사 육성** | 캐릭터의 심상 효과 강화 | 메타 | 영구 |
| **전술 지원 (모집/연구소)** | 인게임 등장 풀 + 효과 강도 | 메타 | 영구 (인게임 등장 시 자동) |
| **출격 모듈** | 인게임 시스템 자체 강화 | 메타 | 영구 (전장마다 자동 적용) |

각 축은 서로 다른 차원에서 작동하므로 역할 중첩이 없다.

## 2. 재화 체계

본 게임은 재화 종류 폭증을 막기 위해 다음 2종 체계를 원칙으로 한다.

| 재화 카테고리 | 용도 |
|---|---|
| **가챠 재화** | 소환술사 모집 · 전술 지원 모집 (수집형) |
| **강화 재화** | 출격 모듈 강화 · (그 외 강화형 시스템) |

원칙: **"뽑기" 는 같은 재화끼리, "강화" 는 같은 재화끼리.**

### 2.1 소환술사 육성 재화 (미결)

소환술사 육성 (레벨업·승급) 에 필요한 재화 구조는 다음 중 결정 필요:

- 강화 재화로 통합
- 기존의 **심상조각** (중복 가챠 시 획득) 체계를 별도로 유지
- 또는 두 체계 혼합

콘텐츠 디자인 단계에서 확정.

## 3. 가챠 시스템

### 3.1 소환술사 가챠

- 메인 메뉴 → **[모집]** 진입.
- 가챠 재화 사용. 1회 / 10회 모집 선택.

#### 신규 소환술사 획득
- 소환술사 캐릭터 해금. 해당 술사의 심상 효과 사용 가능.
- 덱 빌딩(출전 편성) 화면에서 선택 가능 목록에 추가.

#### 중복 소환술사 획득
- 중복 시 처리 방식은 미결 (심상조각 체계 유지 여부와 연동).

### 3.2 전술 지원 가챠

- 가챠 재화 사용. 잠정 소환술사 가챠와 **별도 배너** 로 운영.
- 신규 전술 지원 해금 또는 보유 전술 지원 강화 자원 획득.

### 3.3 가챠 풀 구조 (미결)

- 소환술사 가챠와 전술 지원 가챠의 풀 분리 정책은 잠정 결정이지만 강하게 확정된 사항은 아니다.
- 소환술사 가챠는 "1명씩" 가챠일 가능성이 높으며, 이 경우 풀 구조가 자연스럽게 달라진다.
- 가챠 확률, 천장 시스템, 픽업 등 세부 사항은 콘텐츠 디자인 단계에서 확정.

## 4. 소환술사 육성

### 4.1 진입 흐름

1. 메인 메뉴 → **[소환술사] / [도감]** 진입
2. 소환술사 목록에서 1명 선택 → 상세 화면 진입

### 4.2 레벨 업

- 레벨 업 시 심상 효과가 상승한다.
  - 예) 같은 심상 소환수의 공격력 20 → 22
- 레벨 상한은 승급에 따라 결정된다.

### 4.3 승급

- 레벨 상한 도달 후 승급 가능.
- 승급에는 해당 술사의 심상조각이 필요 (육성 재화 체계 결정에 따라 변경 가능).
- 승급 시 효과:
  - 심상 효과의 최대치 상승
  - 레벨 상한 상승
  - 시너지 임계치 도달 단계 확장 (예: 2 / 3 / 4 → 2 / 3 / 4 / 5)

## 5. 전술 지원 연구소

### 5.1 진입 흐름

1. 메인 메뉴 → **[전술 지원 연구소]** 진입.
2. 현재 해금한 전술 지원과 적용 상태 확인.

### 5.2 해금 경로

- **가챠 모집** (3.2 참고)
- **캠페인 조건부 해금**:
  - 예: 특정 임무 클리어
  - 예: 특정 심상 계열 소환술사 N명 획득

### 5.3 강화

- 강화 재화를 사용해 보유 전술 지원의 효과 강도를 강화.
- 강화된 전술 지원은 인게임 중 등장 후보 풀에 자동으로 반영된다 (별도 장착 단계 없음).

## 6. 출격 모듈 강화

### 6.1 진입 흐름

1. 메인 메뉴 → **[출격 모듈]** 진입.
2. 해금된 출격 모듈 목록 확인.

### 6.2 해금 경로

- 캠페인 진행 보상 등 조건부 해금.

### 6.3 강화

- **강화 재화** 를 사용해 출격 모듈을 강화.
- 보유한 모든 출격 모듈은 한 전장 시작 시 자동 적용된다 (별도 장착 단계 없음).
- 출격 모듈에는 패시브형 (상시 효과) 과 단발성 액티브형 (특정 조건/타이밍 발동) 이 모두 존재한다.

## 7. 후속 정의 필요 항목

다음은 본 문서의 다음 개정에서 정의해야 할 항목들이다.

- 소환술사 육성 재화 체계 (강화 재화 통합 vs 심상조각 별도)
- 전술 지원 가챠와 소환술사 가챠 풀 분리 정책 확정
- 한 번에 등장하는 소환술사 수 (1명 가챠 vs 여러 명 가챠)
- 중복 소환술사 처리 방식
- 가챠 확률 / 천장 시스템
- 강화 곡선 (레벨업 비용·승급 비용·출격 모듈 강화 비용)
- 캠페인 진행에 따른 재화 획득 곡선
