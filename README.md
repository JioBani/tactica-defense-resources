# tactica-defense-resources

`tactica-defense` 프로젝트에서 사용하는 문서·리소스를 분리 관리하는 저장소입니다.

## 디렉터리 구조

```
tactica-defense-resources/
├── docs/                 # 문서
│   ├── design/           # 게임 설계 / 세계관 문서
│   ├── planning/         # 기획 문서
│   └── meeting-notes/    # 회의록
├── assets/               # 게임 리소스
│   ├── images/           # 이미지 (스프라이트, UI, 컨셉 아트 등)
│   ├── audio/            # 사운드 (BGM, 효과음)
│   └── models/           # 3D 모델
└── references/           # 외부 참고 자료
```

## 관련 프로젝트

- [`tactica-defense`](../tactica-defense) — 메인 게임 프로젝트 (Unity)
